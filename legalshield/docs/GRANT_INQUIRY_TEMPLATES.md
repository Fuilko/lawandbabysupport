【優質案件の導流】
AI が被害者の状況を事前に構造化・分析・証拠整理した上で導流します。
貴会の会員弁護士が受け取るのは「まとまらない相談」ではなく、
「整理済みレポート + 関連法条リスト + 証拠フォルダ」付きの優質案件です。

【初回相談の効率化】
• 従来：30分の初回相談のうち20分が情報収集
• LegalShield 導入後：事前にレポートを閲覧済み → 有意義な30分に集中
• 成約率向上：30-40% → 60-70%

【地域格差の解消】
高知県のような地方で、適切な専門家が見つからない被害者を、
貴会の会員弁護士にデジタルでマッチングします。

## 使い方のポイント

| 項目 | 説明 |
|------|------|
| **件名** | 機関名や助成金名称を含める。審査員が一目で内容がわかるように |
| **冒頭** | 「なぜこの機関に申請したいのか」を明確に述べる（相手の価値観との接続） |
| **数字** | 623K 国法・724K 判例・17類型・34項缺陷——具体的数字が信頼性を生む |
| **ストーリー** | 「誰が・どんな被害を・どう救われるか」を語る。技術だけでは心を動かせない |
| **DEMO** | 3分動画 + ONE_PAGER.html を必ず同封またはリンクを貼る |
| **返信** | 3営業日以内に返信しない場合、1週間後に丁寧なフォローメールを送る |
