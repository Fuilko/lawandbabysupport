## 2. プロジェクト概要（400字以内）

```
日本には年間200万人以上の犯罪被害者・権利侵害被害者がいますが、
70%が「誰に相談すればいいか分からない」と1人で抱え込んでいます。

LegalShield は、AI が被害者の代わりに「緊急対応・証拠保全・法律分析・
戦略シミュレーション・専家転介」の5つの役割を果たす被害者支援システムです。
スマートフォン1台で、24時間・無料・匿名で利用可能。

さらに、「被害届不受理防止機能」により、被害者が警察署に行く際に、
AI生成の法的根拠付き報告書を持参することで、警察による被害届不受理を防ぎます。

日本最大級の法律・社会データ基盤（国法623K件、判例724K件、統計885テーブル）を
構築済み。パートナー（弁護士会・NPO・自治体）との連携を進め、
社会課題解決と持続可能な運営を目指します。
```

## 3. 解決する社会課題と必要性（800字以内）

### 課題① 被害者の孤立
- 年間200万人以上の被害者のうち、**専門家に相談するのはわずか30%**
- 残り70%は「面倒くさい」「怖い」「金がかかる」と放置
- 結果として、二次被害・自殺・貧困化が進む

### 課題② 証拠の消失
- 事件発生後24-72時間が「黄金時間」
- しかし被害者は精神的ショックで証拠保全ができない
- 3日後にはLINE履歴削除・傷が治る・現場が変わる

### 課題③ 警察による被害届不受理
- 被害者が「まとまらない話・証拠なし」で警察署に行くと、**相談として処理され被害届が受理されない**
- 警察の「民事不介入」という言い訳で事件性が認められない
- 被害者は「自分が悪いのかも」と思い込み、諦める

### 課題④ 法律資源の地域格差
- 東京・大阪には多くの法テラス・NPOがあるが、**地方にはほとんどない**
- 交通費・時間の制約で専門家に辿り着けない

### LegalShield が解決するアプローチ
1. **AIトリアージ**: 話せなくても、ボタン選択だけで状況把握
2. **自動証拠保全**: 写真・録音をワンタップで暗号化・時系列整理
3. **被害届不受理防止スクリプト**: 警察訪問前にAIが法的根拠・反論文書を生成
4. **GPS連動最寄り施設検索**: どこにいても最寄りの支援機関が見つかる
5. **多言語・高齢者対応**: シンプルUI・音声入力で利用ハードルを下げる
