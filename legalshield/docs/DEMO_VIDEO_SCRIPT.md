# LegalShield デモ動画脚本（3分間）

## 基本情報
- **目的**: 助成金申請・パートナー開拓・投資家向け
- **長さ**: 180秒（3分間）
- **形式**: スマホ画面キャプチャ + ナレーション
- **対象**: 助成金審査員・弁護士・NPO・投資家
- **語り手**: 落ち着いた女性声音（信頼感・安心感）

---

## Scene 0: オープニング（0:00-0:15 / 15秒）

**映像**: 黒背景 → LegalShield ロゴが淡く浮かび上がる
**BGM**: 静かなピアノの和音
**ナレーション**: 
> 「日本には、年間200万人以上の犯罪被害者・権利侵害被害者がいます。」

**テロップ**: 200万人 — 多くの人が支援を受けられない

**ナレーション**:
> 「しかし、その70%が『誰に相談すればいいかわからない』と、1人で抱え込んでいます。」

**切替**: ロゴ → 被害者の孤立を示す抽象的なグラフィック（1人の人物が周囲の情報に溺れている）

## Scene 1: 問題提示（0:15-0:35 / 20秒）

**映像**: 4つの小画面が同時に表示
- 画面A: 被害者がスマホで「DV 相談」と検索 → 情報が多すぎて途方に暮れる
- 画面B: 被害者が警察署に電話 → 「まず被害届を」と言われ、何を持っていけばいいか分からない
- 画面C: 被害者が証拠の写真を撮る → いつかのフォルダに紛失
- 画面D: 被害者が夜中に泣いている → 誰にも連絡できない

**ナレーション**:
> 「被害者が直面するのは、5つの壁です。誰に相談すればいいか分からない。証拠が時間とともに消えていく。自分にどんな権利があるか知らない。近くに支援機関があるか分からない。そして、1人で抱え込み、心が折れる。」

**テロップ**: 
```
① 誰に相談？  ② 証拠が消える  ③ 権利が分からない  
④ 支援機関が分からない  ⑤ 1人で抱え込む
```

## Scene 3: AI 5 重角色の実演（0:50-2:20 / 90秒）

### 3-1. 緊急応変官（0:50-1:10 / 20秒）
**映像**: アプリ画面。「今、安全な場所にいますか？」という大きなボタン。
被害者が「いいえ」をタップ。
画面が暗転 → 赤い「助けて」ボタンが現れる。

**ナレーション**:
> 「まず、緊急応変官。安全確認から、一鍵での 110 発信、位置情報共有、そして周囲に気づかれない無音モードまで。」

**映像**: 10秒カウントダウンが表示 → キャンセルせず → サイレントSMS送信完了の表示
「支援メッセージを送信しました」

**テロップ**: 🚨 緊急応変官 — 無音・匿名・即時

---

### 3-2. 証拠保全官（1:10-1:30 / 20秒）
**映像**: 被害者が「写真を撮る」をタップ。スマホカメラが起動。
傷の写真を撮影 → 自動的にタイムスタンプ・GPS・ハッシュ値が付与されるアニメーション

**ナレーション**:
> 「証拠保全官。写真、録音、書類をアップロードするだけで、AI が自動的に時系列整理、分類、暗号化を行います。裁判所に提出可能な保全証明書も自動生成。」

**映像**: フォルダが自動生成される様子
```
LegalShield_Case_001/
├── 02_証據/
│   ├── 照片/
│   ├── 訊息截圖/
│   └── 醫療記錄/
└── 05_証明書/
    └── evidence_certificate.json
```

**テロップ**: 📁 証拠保全官 — SHA-256 + タイムスタンプ + 暗号化

### 3-3. 法律分析師（1:30-1:55 / 25秒）
**映像**: 被害者がマイクボタンをタップ。泣きながら話す。
> 「えっと…昨日…夫が…酒を飲んで…急に…私を…殴って…」

画面に文字起こしがリアルタイムで表示される。
話が終わると、AI が状況を整理：
```
事件日時: 昨日 夜間
加害者: 夫（配偶者）
暴力タイプ: 顔面殴打（飲酒）
現在: 避難中
懸念: 子供が自宅に残っている
緊急度: HIGH
```

**ナレーション**:
> 「法律分析師。感情にまみれた被害者の言葉から、AI が自動的に事実を抽出・整理。そして、623,000 件の国法と 724,000 件の判例を瞬時に検索。」

**映像**: 関連法條が次々と表示される
- 配偶者暴力防止及び被害者保護法
- 刑法 204 条（傷害罪）
- 民法 709 条（損害賠償）

**テロップ**: ⚖️ 法律分析師 — 623K 国法 + 724K 判例 RAG 検索

## 技術的撮影ガイド

### 必要な機材
- iPhone または Android（スクリーン録画機能付き）
- マイク（AirPods Pro のマイクでも可）
- 三脚またはスマホスタンド（画面ブレ防止）

### スクリーン録画方法
**iPhone**: 設定 → コントロールセンター → 画面収録 を追加 → 録画開始
**Android**: クイック設定パネル → 画面収録

### ナレーション録音
1. 静かな部屋で録音（風通しの音、冷房音に注意）
2. ナレーション原稿を印刷し、自然な間を入れながら読む
3. 1 テイクで完璧を狙わず、3-4 テイク録音して後で編集

### 編集（無料ツール）
- **CapCut**（スマホ版）: カット・テロップ・BGM 挿入が簡単
- **DaVinci Resolve**（PC版）: プロ級、無料
- **iMovie**（Mac/iPhone）: 初心者向け

### BGM ソース（著作権フリー）
- **YouTube Audio Library**: 無料、商用利用可
- **フリー BGM 素材「 MusMus 」**: 日本のクリエイター向け
- **Artlist**（有料）: 質が高い
