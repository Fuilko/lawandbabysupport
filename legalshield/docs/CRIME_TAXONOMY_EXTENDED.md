### 6. サイバー犯罪・ネットいじめ

#### 被害者像
- 主に10-30代。SNS利用者。特定の理由（容姿・思想・交友関係）で標的にされる。
- 二次被害: 不眠・不安障害・自閉・自殺念慮。

#### 加害者側写（Perpetrator Profile）
| 項目 | 特徴 |
|------|------|
| **年齢層** | 10-40代（若年層に多い） |
| **性別比** | 男性7:女性3（いじめ扇動者は男性が多いが、女性同士の排除も激しい） |
| **心理的特徴** | 自己肯定感の低さ、集団への帰属欲求、見下すことで優越感を得る |
| **SNS使用** | 匿名アカウント・複数アカウント・ burner accounts |
| **常用狡弁** | 「ただの冗談」「お前が悪いから」「デマでもいい、面白ければ」「世間は私の味方」 |
| **弱点** | 匿名性に依存。実名・顔がバレると怯える。法律知識がない。 |

#### 狡弁と反論
| 加害者の狡弁 | 被害者の反論 |
|------------|------------|
| 「ただの冗談だ、超えてる」 | 繰り返しの中傷は『侮辱罪』に該当します。相手の気持ちを考えない「冗談」は加害です。 |
| 「お前が先に悪いことした」 | たとえ被害者に過失があったとしても、それを理由に中傷・拡散することは別の犯罪です。 |
| 「デマでも、誰かが言ってたから」 | 拡散責任は拡散者にもあります。『名誉毀損』の共同不法行為が成立します。 |
| 「匿名だからバレない」 | IPアドレス・端末情報から特定は可能です。捜査機関の開示請求で即座に特定されます。 |
| 「発言の自由だ」 | 発言の自由には責任が伴います。他人の権利を侵害する発言は保護されません。 |

#### 証拠保全チェックリスト
- [ ] 全投稿のスクリーンショット（URL・日時・投稿者名を含む）
- [ ] スレッドの全文キャプチャ
- [ ] IPアドレス・位置情報の記録（可能な場合）
- [ ] 被害者自身の精神的影響を記録（日記・医療機関受診記録）
- [ ] フォロワー数・拡散数（拡散範囲の証拠）

#### AI対話腳本（被害者→AI）
> 「SNSで知らない人から毎日のように罵倒されて、もう耐えられない」

**AI応答**:
「それは本当に辛い状況です。あなたは1人ではありません。
まず、**今日から全ての投稿をスクリーンショットで保存**してください。
次に、**匿名のまま警察に相談**できます（サイバー犯罪相談窓口）。

あなたの場合、『侮辱罪』または『名誉毀損罪』の可能性があります。
相手が『ただの冗談』と言っても、繰り返しの中傷は犯罪です。

**今すぐできること**:
1. ブロック・ミュート
2. スクリーンショット保存
3. サイバー犯罪相談窓口（#9110）へ連絡

あなたの気持ちは正しいです。続けましょうか？」

### 7. ストーカー・つきまとい

#### 被害者像
- 主に女性（80%以上）。元交際相手・職場の人・SNSで知り合った人から。
- 二次被害: 極度の不安・不眠・場所を変える・職場退職。

#### 加害者側写
| 項目 | 特徴 |
|------|------|
| **年齢層** | 20-50代 |
| **性別比** | 男性9:女性1 |
| **心理的特徴** | 執着型愛着障害、ナルシシスト的パーソナリティ、拒否を受け入れられない |
| **行動パターン** | 自宅周辺徘徊、職場待ち伏せ、SNS監視、連絡先への接触、プレゼント送付 |
| **常用狡弁** | 「心配だから」「愛してるから」「お前が私を惑わせた」「別れたのはお前のせい」 |
| **弱点** | 被害者からの関心が完全に消えると焦燥。法的措置に弱い。 |

#### 狡弁と反論
| 加害者の狡弁 | 被害者の反論 |
|------------|------------|
| 「心配だから連絡してるだけ」 | 一方的な連絡・待ち伏せは『ストーカー行為』です。心配を理由にした執着は犯罪です。 |
| 「愛してるから」 | 愛情を理由に相手の意思に反する接触を続けることは、愛ではなく支配です。 |
| 「別れたのはお前が悪い」 | 別れの理由は何であれ、相手が関係を終了した以上、追う権利はありません。 |
| 「ただの偶然、道が同じだけ」 | 何度も『偶然』が続くことはありません。位置情報・防犯カメラで証明できます。 |
| 「お前が私をこうさせた」 | 自分の行為の責任を他人に押し付けることは、加害者の典型的心理です。 |

#### 証拠保全チェックリスト
- [ ] 全メッセージ・通話履歴のスクリーンショット
- [ ] 待ち伏せの日時・場所記録
- [ ] 防犯カメラ・ドライブレコーダー映像
- [ ] プレゼント・郵便物の写真
- [ ] 第三者（家族・友人・職場）への接触記録
- [ ] GPS・位置情報の記録

#### AI対話腳本
> 「元彼が自宅の前に立って待ってる。もう何ヶ月も続いてる」

**AI応答**:
「それは本当に恐ろしい状況です。あなたの恐怖は正当です。
ストーカー行為は、**『つきまとい』だけでなく、『監視』『待ち伏せ』も含まれます**。

**今すぐできること**:
1. **警察に相談**（被害届・保護命令申請）
2. **ストーカー相談ダイヤル #8000**
3. **身の回りの人に状況を伝える**（職場・家族・友人）
4. **自宅の防犯対策**（カメラ・ドアロック・ライト）

保護命令が出れば、加害者はあなたに近づけなくなります。
一緒に準備しましょう。」

### 8. 薬物被害・依存

#### 被害者像
- 年齢層が広い（10代〜60代）。初回は「体験」「友人から勧められて」が多い。
- 二次被害: 身体依存・精神障害・犯罪への巻き込み・家庭的・社会的破滅。

#### 加害者側写
| 項目 | 特徴 |
|------|------|
| **年齢層** | 20-50代 |
| **性別比** | 男性6:女性4 |
| **タイプA: 提供者** | 「楽にしてあげた」「みんなやってる」と勧誘。金銭利益目的。 |
| **タイプB: 組織的** | ヤクザ・ギャング関連。被害者を「末端販売者」に使う。 |
| **タイプC: 共犯者** | 自身も使用者。被害者を引きずり込むことで自己正当化。 |
| **常用狡弁** | 「1回くらい大丈夫」「自分で選んだんだ」「楽にしてあげた」「依存は意志が弱いから」 |
| **弱点** | 金銭目的が明確な場合、資金流れが追跡可能。 |

#### 狡弁と反論
| 加害者の狡弁 | 被害者の反論 |
|------------|------------|
| 「1回くらい大丈夫、体験だ」 | 覚せい剤・大麻・MDMAなど、多くの薬物は1回でも依存リスクがあります。勧誘者には責任があります。 |
| 「自分で選んだんだから」 | 被害者が18歳未満、または精神的に不安定な状況であれば、同意能力に疑問があります。 |
| 「依存は意志が弱いから」 | 薬物依存は医学的な病気です。治療が必要であり、個人の「意志」だけで克服できるものではありません。 |
| 「楽にしてあげたのに」 | 一時的な快感を与えることで被害者を依存させ、人生を破壊する行為は「援助」ではなく「害行」です。 |
| 「知らなかった、危険だとは」 | 薬物の危険性は常識です。「知らなかった」は免責になりません。 |

#### 証拠保全チェックリスト
- [ ] 薬物の写真（容器・包装・残量）
- [ ] 購入・譲渡の金銭履歴
- [ ] メッセージ・通話録音
- [ ] 医療機関の診断書（依存症・健康被害）
- [ ] 近隣・友人の証言
- [ ] 提供場所の情報

### 9. ヘイトクライム・差別

#### 被害者像
- 少数民族・外国人・障害者・LGBTQ+・宗教的信条の違う人。
- 二次被害: 社会的孤立・精神的創傷・就職・住居への差別。

#### 加害者側写
| 項目 | 特徴 |
|------|------|
| **年齢層** | 20-60代（ネット上では10代も） |
| **性別比** | 男性8:女性2 |
| **心理的特徴** | 外集団嫌悪、内的な劣等感の補償、イデオロギーへの過度な同調 |
| **行動パターン** | SNS拡散、職場での差別、住居侵入、暴言・暴力、集会 |
| **常用狡弁** | 「日本を守るため」「事実を言ってるだけ」「差別じゃなく区別」「被害者が過敏」 |
| **弱点** | 論理的に崩されると感情的になる。法的知識がない。 |

#### 狡弁と反論
| 加害者の狡弁 | 被害者の反論 |
|------------|------------|
| 「日本を守るため」 | 日本国憲法は人種差別を禁じています（14条）。「愛国心」を理由に差別を正当化することは憲法違反です。 |
| 「事実を言ってるだけ」 | たとえ事実であっても、それを悪意を持って拡散し、差別的な文脈で使用することは『名誉毀損』です。 |
| 「差別じゃなく区別」 | 人の価値を生まれや外見で決めることは、憲法が保障する「幸福追求権」「人格権」を侵害します。 |
| 「被害者が過敏」 | 差別的言動が相手に苦痛を与えることは、加害者の「意図」ではなく「結果」で判断されます。 |
| 「言論の自由」 | 言論の自由は、人種・民族・信条を根拠にした憎悪を煽る行為には適用されません。 |

### 10. 売買春・人身取引

#### 被害者像
- 主に女性・子ども。経済的困窮・DV避難・外国籍・障害者が標的になりやすい。
- 二次被害: PTSD・性的トラウマ・性感染症・社会的烙印。

#### 加害者側写
| 項目 | 特徴 |
|------|------|
| **年齢層** | 30-60代 |
| **性別比** | 男性9:女性1（女性は「斡旋者」として参画する場合も） |
| **心理的特徴** | 金銭至上主義、他者の身体的自律権を軽視、組織的支配欲 |
| **タイプ** | 個人斡旋者・スカウト・風俗経営者・国際人身取引組織 |
| **常用狡弁** | 「本人が選んだ仕事」「金を稼げて助けてあげた」「違法じゃない」「嘘を言ってる」 |
| **弱点** | 金銭流れが追跡可能。被害者の証言が一致すれば組織全体が崩壊。 |

#### 狡弁と反論
| 加害者の狡弁 | 被害者の反論 |
|------------|------------|
| 「本人が選んだ仕事」 | 経済的困窮・DV・人身拘束など、自由な選択ができない状況での「同意」は無効です。 |
| 「金を稼げて助けてあげた」 | 被害者の身体的・精神的健康を犠牲にした金銭獲得は「搾取」であり「援助」ではありません。 |
| 「18歳以上だから合法」 | 18歳以上であっても、脅迫・詐欺・人身拘束を用いた売春は刑法226条の2（人身売買）に該当します。 |
| 「嘘を言ってる、金目的」 | 被害者が金銭目的であっても、それを利用した搾取行為の違法性は消えません。 |
| 「知らなかった、本人が言ってた」 | 被害者の年齢・状況を確認しなかったことは、加害者側の過失として問われます。 |

### 11. 監禁・監視

#### 加害者側写
| 項目 | 特徴 |
|------|------|
| **心理的特徴** | 極度の支配欲、所有欲、不安定な自己同一性 |

|------------|------------|

| 「家族が危ない」 | 家族への危害をちらつかせることは『脅迫罪』です。被害者をコントロールする手段です。 |

## 加害者通用狡弁パターン（横断的分析）

全犯罪タイプを横断して見られる加害者の共通狡弁パターン:

### パターンA: 被害者責任転嫁
> 「お前が悪い」「お前が先に」「お前がそうさせた」
- **心理**: 自己の行為を正当化するため、被害者に原因があると認識する。
- **反論**: 被害者に過失があっても、それを理由に違法行為を行うことは許されません。

### パターンB: 愛情・友情・援助の装い
> 「愛してるから」「心配だから」「助けてあげた」
- **心理**: 支配・搾取を「善意」の名目で隠蔽する。
- **反論**: 相手の意思に反する行為は、どんな「善意」の名目も犯罪を正当化しません。

### パターンC: 軽視・矮小化
> 「ただの冗談」「大げさ」「みんなやってる」
- **心理**: 被害者の苦痛を認めると自分の悪さを認めなければならないため、矮小化する。
- **反論**: 被害者が苦痛を感じたことは客観的事実です。加害者の「ただの」という言葉は無力です。

### パターンD: 権利の濫用主張
> 「発言の自由」「日本を守る」「正義」
- **心理**: 憲法・法律の概念を自らの都合に歪曲して使用する。
- **反論**: すべての権利には責任が伴います。他人の権利を侵害する行為は、どんな大義名分も保護されません。

### パターンE: 無知の主張
> 「知らなかった」「気づかなかった」「危険だとは」
- **心理**: 故意性を否定することで責任を回避しようとする。
- **反論**: 「知らなかった」は、常識的に知るべきだったこと・確認義務があったことについては免責になりません。

### 16. 痴漢・公共交通内性犯罪

#### 被害者像
- 主に女性（10代〜40代）。通勤・通学中の電車・バスが圧倒的に多い。
- 混雑時間帯（朝7-9時、夕方17-19時）が高危険時間帯。
- 二次被害: 恐怖（電車に乗れなくなる）・自己嫌悪・不眠・社会的場所への回避。

#### 加害者側写（Perpetrator Profile）
| 項目 | 特徴 |
|------|------|
| **年齢層** | 20-60代（幅広い） |
| **性別比** | 男性が圧倒的多数（99%+） |
| **心理的特徴** | 触物癖（frotteurism）・恋物癖、共感能力の欠如、匿名性への依存 |
| **行動パターン** | 混雑車両を狙う、逃げ場のない位置（ドア際・座席間）を選定、短時間で実行・離脱 |
| **再犯率** | **極めて高い**。逮捕歴ありの常習犯が大部分。 |
| **常用狡弁** | 「電車が混んでて偶然触れた」「何もしてない」「女のほうが悪い」「痴漢冤罪」 |
| **弱点** | 常習性があるため、パターン化している。防犯カメラ・目撃者が決定的。 |

#### 狡弁と反論
| 加害者の狡弁 | 被害者の反論 |
|------------|------------|
| 「電車が混んでたから偶然」 | 混雑は痴漢の『方便』です。明確な触覚・圧力がかかっていたら、偶然ではありません。 |
| 「何もしてない、冤罪だ」 | 被害者がわざわざ騒ぐメリットはありません。女性が痴漢と認識した時点で、それは犯罪です。 |
| 「彼女が悪い、服がエロい」 | 服装は加害の理由になりません。痴漢は加害者の欲望の問題であり、被害者の服装とは無関係です。 |
| 「怒鳴るな、大げさだ」 | 被害者の恐怖・不快感の大きさは、被害者が決めます。加害者が「大げさ」と決めつける権利はありません。 |
| 「前科があるわけじゃない」 | 痴漢は常習犯が多く、多くの場合は初犯ではなく**初めて逮捕された**だけです。 |

#### 証拠保全チェックリスト
- [ ] 駅・車内の防犯カメラの有無確認（駅員・警察にリクエスト）
- [ ] 目撃者の名前・連絡先
- [ ] 被害者の服装記録（痴漢が触った場所の特定のため）
- [ ] 加害者の特徴（身長・服装・髪型・バッグ・靴）
- [ ] 事件発生時刻・駅名・車両番号
- [ ] その日の電車混雑状況（他の乗客の証言を含む）

#### AI対話腳本（被害者→AI）
> 「電車で隣の人に触られて、気持ち悪かった」

**AI応答**:
「それは本当に不快な経験でした。あなたの感覚は正しいです。
痴漢は**刑法176条（強制わいせつ）**に該当する可能性があります。

**今すぐできること**:
1. **駅の駅員・警察に通報**（次の駅で降りて通報）
2. **加害者の特徴をメモ**
3. **被害を受けた部位の服装をそのままにして保存**
4. **防犯カメラの証拠保全を依頼**

次の駅で駅員さんに『痴漢に遭いました』と伝えてください。

### 17. 公然わいせつ・暴露狂

#### 被害者像
- 全年齢層。特に**子供・青少年**が標的になりやすい（公園・学校周辺）。
- 女性被害者が圧倒的多数。
- 二次被害: 恐怖・トラウマ・場所回避（公園・暗い道など）。

#### 加害者側写（Perpetrator Profile）
| 項目 | 特徴 |
|------|------|
| **年齢層** | 20-60代 |
| **性別比** | 男性が圧倒的多数 |
| **心理的特徴** | **露体癖（エキシビショニズム）**、被害者の驚き・恐怖反応が性的興奮のトリガー、社会的コミュニケーション能力の欠如 |
| **行動パターン** | 公園・駅・学校周辺・駐車場・マンション駐輪場。被害者を選定して接近。車から露出することも多い。 |
| **常習性** | **再犯率が非常に高い**。同じ場所・時間帯を繰り返す傾向あり。 |
| **常用狡弁** | 「着替え中だった」「気づかなかった」「子供だから何もわからないだろう」「開放的な場所の定義が違う」 |
| **弱点** | 場所・時間帯が固定されているためパターン化しやすい。防犯カメラが効果的。 |

#### 狡弁と反論
| 加害者の狡弁 | 被害者の反論 |
|------------|------------|
| 「着替え中だった、見られただけ」 | 公共の場で意図的に露出したことは、『公然わいせつ』です。着替えの理由は成立しません。 |
| 「気づかなかった、誰もいないと思った」 | 公園・道路等は常に人がいる可能性がある場所です。「誰もいないと思った」は免責になりません。 |
| 「子供だから何もわからない」 | 子供だからこそ、恐怖の影響は深刻です。児童への公然わいせつは重く処罰されます。 |
| 「大げさに取り上げるな」 | 被害者の恐怖・不快感の大きさは被害者が決めます。加害者が決めつけることはできません。 |
| 「冤罪だ、服を直してただけ」 | 動作・場所・状況から、意図的な露出か偶発的な動作かは判断可能です。 |

#### 証拠保全チェックリスト
- [ ] 防犯カメラ（公園・駅・建物）のリクエスト
- [ ] 加害者の特徴（身長・体型・服装・髪型・靴・車種・ナンバー）
- [ ] 目撃者の連絡先
- [ ] 現場の写真・動画（自分が安全な場所から撮影）
- [ ] 事件発生時刻・場所（正確なGPS位置）
- [ ] 加害者が逃走した方向

#### AI対話腳本（被害者→AI）
> 「公園で突然、知らないおじさんが服を脱いでた。びっくりして逃げた」

**AI応答**:
「それは本当に恐ろしい体験でした。あなたはよく逃げました。
それは**刑法174条（公然わいせつ）**に該当する可能性が高いです。

**今すぐできること**:
1. **110番通報**（今すぐ、または安全な場所から）
2. **加害者の特徴をメモ**（車があればナンバーも）
3. **現場の防犯カメラを確認**
4. **近くの交番・警察署へ**（目撃者としての証言を記録）

そのおじさんは**同じ場所・時間帯を繰り返す**ことが多いです。
通報することで、次の被害者を防げます。
一緒に準備しましょうか？」
