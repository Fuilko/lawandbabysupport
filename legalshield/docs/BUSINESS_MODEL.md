# LegalShield ビジネスモデル・収益化戦略

## 核心理念

> **「公益を先に、収益は自然に。」**

被害者支援の本質は「誰も1人にはしない」社会貢献にあり、その信頼とデータ資產が長期的な収益を生む。

---

## 収益ストリーム（4本柱）

### 柱 1: B2B 専家導流（Lead Generation）

**モデル**: AIが被害者の状況を構造化・分析した後、「弁護士・NPO・支援機関」へシームレスに繋ぐ。専家は「まとまらない話の被害者」ではなく、「整理済み・分析済みの優質案件」を受け取る。

**なぜ専家が支払う価値があるか**
| 課題 | 従来 | LegalShield 導入後 |
|------|------|-------------------|
| 初回相談 | 30分中20分が情報収集 | 整理済みレポートを事前閲覧 |
| 案件質 | まとまらない話、証拠不足が多い | 証拠フォルダ・法條リスト付き |
| 成約率 | 30-40% | 60-70%（ニーズ明確なため） |
| 時間単価 | 初回30分が無料or低額 | 有意義な30分に集中 |

**収益構造**
```
月間アクティブ被害者: 1,000名
    ↓ 専家マッチング率: 30% (300名)
    ↓ 専家側月額サブスクリプション: ¥30,000/事務所
    ↓ 導流手数料: 成約時 ¥5,000/件

想定収益:
  弁護士事務所 50 件 × ¥30,000 = ¥1,500,000/月
  導流手数料 300件 × 50%成約 × ¥5,000 = ¥750,000/月
  小計: ¥2,250,000/月 = ¥27,000,000/年
```

**契約タイプ**
| プラン | 月額 | 含まれるもの |
|--------|------|------------|
| ライト | ¥10,000 | 月5件の整理済み案件閲覧 |
| スタンダード | ¥30,000 | 月20件 + 優先マッチング + 統計レポート |
| プレミアム | ¥100,000 | 無制限 + API連携 + ホワイトラベル |

### 柱 2: B2G GovTech SaaS（自治体・政府向け）

**モデル**: LegalShield のバックエンド（1.36M 向量 DB + AI 5 重角色）を SaaS 化し、自治体の「児童相談所」「DV 支援センター」「消費者センター」に販売。

**なぜ自治体が買う価値があるか**
| 自治体の課題 | LegalShield の解決 |
|------------|-------------------|
| 相談記録の非構造化 | AI が自動構造化・分類 |
| 後続フォロー漏れ | AI が期限管理・自動リマインド |
| 専門家不足 | AI が初動対応を代行 |
| 年次報告書作成 | 自動統計・可視化ダッシュボード |
| 緊急時対応遅延 | 24時間 AI 自動トリアージ |

**収益構造**
```
導入自治体: 47 都道府県 + 市区町村 100 箇所
    ↓ 都道府県単価: ¥3,000,000/年
    ↓ 市区町村単価: ¥500,000/年

想定収益:
  都道府県 20 箇所 × ¥3,000,000 = ¥60,000,000/年
  市区町村 50 箇所 × ¥500,000 = ¥25,000,000/年
  小計: ¥85,000,000/年
```

**SaaS 機能モジュール**
| モジュール | 説明 |
|-----------|------|
| 相談トリアージ AI | 自治体職員が電話・対面相談時にAIがリアルタイム補助 |
| 自動レポート生成 | 月次・年次報告書をワンクリック生成 |
| ヒートマップ | 地域別被害発生マップ（プライバシー保護済） |
| 専家マッチング | 管内の弁護士・NPO リスト連携 |

### 柱 3: データ洞察レポート（Data Analytics）

**モデル**: 匿名化・集計化された被害データのトレンド分析レポートを、保険会社・学術機関・メディア・投資家に販売。

**データ資產の価値**
| データセット | 販売対象 | 価値 |
|-----------|---------|------|
| 全国 DV 発生トレンド | 保険会社 | 保険商品設計、リスク評価 |
| 消費者被害手法の進化 | 金融機関 | 詐欺対策システム改善 |
| 地域別法律資源ギャップ | 政府・智庫 | 政策提言、予算配分 |
| 職場ハラスメント実態 | 企業 HR | 教育資料、リスク管理 |
| 季節・時間帯パターン | メディア | 報道特集、啓発企画 |

**収益構造**
```
年間レポート販売: 20 件
    ↓ 単価: ¥1,000,000-5,000,000/レポート
    ↓ カスタム分析: ¥10,000,000/件

想定収益: ¥50,000,000-100,000,000/年
```

**倫理ガードレール**
- **完全匿名化**: 個人識別不可能な形でのみ販売
- **集計最小単位**: 市区町村単位、個別事例は含めない
- **倫理審査**: 学術機関連携、IRB（倫理審査委員会）承認
- **オプトアウト**: ユーザーがデータ利用に同意しない場合、完全除外

### 柱 4: 被害者保険・金融連携

**モデル**: 保険会社・金融機関と連携し、被害者向けの「法的手続費用保険」「緊急資金貸付」を仲介。

**商品例**
| 商品名 | 内容 | ターゲット |
|--------|------|-----------|
| 法律費用保険 | 弁護士費用を月 ¥1,000 でカバー（年間上限 ¥300万） | 一般消費者 |
| DV 緊急脱出資金 | 無利子・無担保で最大 ¥100 万を即時融資 | DV 被害者 |
| 性暴力治療費保険 | 医療費・心理諮商費をカバー | 全女性対象 |
| 消費者被害補償 | 詐欺被害額の一部を補償 | 高齢者向け |

**収益構造**
```
仲介手数料: 保険料の 10-20%
想定収益: ¥20,000,000-50,000,000/年
```

## 収益化のフェーズ設計

### Phase 0: 純粋公益（現在〜1年後）
- 被害者への利用: **完全無料**
- 収益源: 助成金のみ
- 目的: ユーザー獲得・データ蓄積・信頼構築

### Phase 1: パートナー収益化（1年〜2年後）
- 弁護士会・NPO へのサブスクリプション開始
- 自治体パイロット（1-2 箇所）
- 被害者は依然として無料

### Phase 2: 多角収益化（2年〜3年後）
- B2B + B2G + データレポート 同時展開
- 保険・金融連携開始
- 一部有料機能（専家版 Pro）

### Phase 3: 持続可能運営（3年〜5年後）
- 助成金依存から脱却
- 営利部門と非営利部門の分離検討
- 海外展開（台湾・韓国・東南アジア）
