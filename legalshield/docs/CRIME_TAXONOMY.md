# LegalShield 犯罪分類與 AI 對應處理矩陣

## 核心理念
每種犯罪類型有不同的「黃金時間」「證據特性」「法律路徑」「心理創傷模式」。AI 必須針對每種類型提供定制化流程。

---

## 第一層：暴力犯罪 (Violent Crimes)

### 1.1 配偶者暴力 (DV: Domestic Violence)

**特徵**
- 反覆性：暴力循環（緊張→爆發→和解→蜜月→再緊張）
- 隔離性：加害者切斷受害者社會連結
- 經濟控制：沒有獨立經濟來源

**黃金時間**
- 受傷後 72 小時內：醫療驗傷、拍照
- 威脅訊息：立即截圖，不要刪除
- 保護命令：隨時可申請，但緊急時更快

**AI 提供資訊**
```
□ 即時安全確認：您是否處於安全環境？
□ 緊急避難所：最近的 婦人保護施設 / DV シェルター
□ 醫療機構：可以驗傷的指定醫院
□ 保護命令申請：家庭裁判所流程、所需文件
□ 經濟自立：生活保護、一時金、就業支援
□ 親權問題：離婚時的親權、養育費、面會交流
```

**證據清單**
| 優先度 | 證據類型 | 保存方式 | 注意事項 |
|--------|---------|---------|---------|
| ★★★ | 診斷書 | 醫院原件 + 影本 | 記載「外傷」「挫傷」等 |
| ★★★ | 受傷照片 | 手機拍照 + 雲端 | 臉部入鏡、時間戳 |
| ★★★ | 威脅訊息 | 截圖 + 錄屏 | LINE、SMS、Mail |
| ★★☆ | 報案記錄 | 警察署受理番號 | 即使不立案也要記錄 |
| ★★☆ | 證人 | 鄰居、親友證言 | 目擊時間地點 |
| ★☆☆ | 財務記錄 | 銀行明細 | 經濟控制證據 |

**對話輔助腳本**
```
[與警察對話腳本]
「私は配偶者から暴力を受けています。
 今日○○時、○○で○○されました。
 診断書があります。保護命令を検討しています。」

[與醫生對話腳本]
「DV の被害です。診断書に『外傷の原因：
 配偶者による暴力』と明記してください。
 これは裁判で使用します。」

[與弁護士對話腳本]
「離婚と損害賠償を検討しています。
 子供が○歳で、私は専業主婦です。
 資産状況は○○です。」
```

### 1.2 性暴力 (Sexual Violence)

**特徵**
- 高度創傷：羞恥、自責、否認
- 證據脆弱：精液、DNA 隨時間降解
- 二次傷害：司法程序可能再次創傷

**黃金時間**
- **最重要：不洗澡、不換衣服、不上廁所**
- 24 小時內：法醫學檢查
- 72 小時內：HIV 預防投藥
- 1 週內：心理諮商介入

**AI 提供資訊**
```
⚠️  緊急提醒：
  1. まだシャワーを浴びていない場合は絶対に浴びないでください
  2. 衣類は紙袋に入れて保存（ビニール袋は避ける）
  3. トイレに行く場合、排尿せず病院へ

□ 指定病院：法医学検査ができる病院リスト
□ ワンストップ支援：#8898
□ HIV 予防：72時間以内の PEP 投薬
□ 緊急避妊：120時間以内
□ 心理諮商：創傷後ストレス障害（PTSD）予防
```

**證據清單**
| 優先度 | 證據類型 | 保存方式 | 注意事項 |
|--------|---------|---------|---------|
| ★★★ | 衣類 | 紙袋、乾燥した場所 | ビニールはカビの原因 |
| ★★★ | 体内遺留物 | 病院採取 | 24時間以内が最適 |
| ★★★ | 現場照片 | 場所、傷害 | 人が写らない範囲で |
| ★★☆ | 通訊記錄 | 犯人とのやり取り | 合意の有無が争点 |
| ★★☆ | 目撃者 | 同行者、店員 | 被害者の状態確認 |
| ★☆☆ | 位置情報 | 手機 GPS | 移動経路 |

**心理支援腳本**
```
「これはあなたのせいではありません。
 誰にも言えない気持ちは自然です。
 一人で抱えなくても大丈夫です。
 私たちが一緒に対処法を考えます。」
```

### 1.3 児童虐待 (Child Abuse)

**特徵**
- 隱蔽性：兒童無法表達或恐懼報復
- 權力不對等：親權者 vs 完全依存的兒童
- 長期影響：發展障礙、成年後創傷

**黃金時間**
- 即時：兒童安全確保（#7119）
- 發現後 24 小時：詳細記錄（兒童發言、傷害）
- 1 週內：兒童心理諮商

**AI 提供資訊**
```
⚠️  児童の安全が最優先

□ 通報先：児童相談所 #7119（24時間）
□ 緊急時：警察 110
□ 通報者保護：匿名可能、職務上の通報は義務
□ 臨時保護：児童相談所が 48時間以内に決定
□ 親権停止：家庭裁判所
```

**通報者證據清單**
| 優先度 | 證據類型 | 保存方式 | 注意事項 |
|--------|---------|---------|---------|
| ★★★ | 兒童發言 | 逐字記錄、時間 | 「誰が」「何を」「いつ」 |
| ★★★ | 傷害照片 | 自然光、全身 | 日期標籤 |
| ★★☆ | 學校記錄 | 成績下降、態度變化 | 教師觀察 |
| ★★☆ | 醫療記錄 | 反覆的受傷 | 不同醫院的記錄 |
| ★☆☆ | 家庭環境 | 居家訪問記錄 | 兒童相談所が実施 |

## 第二層：經濟犯罪 (Economic Crimes)

### 2.1 消費者被害 (Consumer Fraud)

**類型細分**
| 類型 | 特徵 | 黃金時間 |
|------|------|---------|
| 振り込め詐欺 | 電話で親族装う | 振込後即時（銀行に停止依頼） |

| 悪徳商法 | 高額契約、クーリングオフ対象 | 契約後 8 日間 |
| キャッチセール | 路上での強引な勧誘 | 即時 |
| サブスク陷阱 | 解約困難な定期購入 | 解約期限前 |

**AI 提供資訊**
```
□ 消費者ホットライン: 188（平日 9-17時）
□ 国民生活センター: 0120-940-828
□ クーリングオフ: 契約書面到達後 8 日間
□ 銀行停止: 振込後すぐに銀行へ電話
□ 被害者支援基金: 都道府県による給付
```

## 電子檔保存ガイドライン

### 證據保存の鉄則「3-2-1 ルール」
- **3** 箇所にコピー
- **2** 種類のメディア（ローカル + 雲端）
- **1** 箇所はオフライン（USB 等）

### 保存手順
```
1. 撮影/取得
   ↓
2. 即時アップロード（LegalShield Vault）
   → SHA256 ハッシュ生成
   → GPS 座標記録
   → タイムスタンプ付与
   ↓
3. 暗号化
   → AES-256 でローカル暗号化
   → パスワードは別途管理
   ↓
4. バックアップ
   → 信頼できる家族/友人に共有
   → クラウドストレージ（iCloud/Google Drive）
   → オフライン USB
   ↓
5. 証明書発行
   → 保全証明書（時間・ハッシュ・位置）
   → PDF で出力
```

### 各ファイルタイプの対応
| 類型 | 保存方法 | 注意事項 |
|------|---------|---------|
