## 2. 不足・補強すべき証拠（優先順位付き）

### 🚨 優先度 ★★★（6/15 までに必須）

#### G1. mapry 法人登記事項証明書

**何のため**: 訴訟管轄（本店所在地）+ 代表者名（代表訴訟当事者）+ 法人番号確定
**取得方法**:
1. 法務省「登記情報提供サービス」: https://www.touki-kyoutaku-online.moj.go.jp/
2. オンライン申請 → 即時取得（コンビニ印刷可）
3. 費用: **¥600**（オンライン）/ ¥600（窓口）
**所要時間**: 30 分

**作業手順**:
```
1. 利用者登録（無料）
2. 「商業・法人」検索
3. 検索条件「mapry」+ 都道府県
4. 該当法人選択 → クレジットカード決済 ¥600
5. PDF ダウンロード（数分）
```

**確認すべき情報**:
- 本店所在地（→ 管轄裁判所）
- 代表取締役（→ 訴状の被告代表者欄）
- 設立年月日 / 資本金 / 事業目的
- 役員一覧（責任追及範囲）

#### G2. 差別・脅迫メール フォーマット化

**何のため**: 法務省人権擁護局申立 + 民事不法行為立証 + 刑事 222 条証拠

**現状**: `.eml` ファイル 7 件あり（生形式）
**作業**:
1. 各 .eml をテキスト抽出（メーラーで表示 → 全文コピー）
2. **ヘッダ情報**全文保存（Received: / From: / To: / Date: / Message-ID:）
3. **SHA256 ハッシュ計算**（改竄防止証拠）
4. 整理表作成: 日付 / 差出人 / 件名 / 主要文言 / 該当法令

**所要時間**: 1-2 時間

**ハッシュ計算コマンド（PowerShell）**:
```powershell
Get-FileHash 'E:\mapry\差別メール1.eml' -Algorithm SHA256
```

**フォーマット例**:
```
甲第6号証の1
日付:        2026年2月XX日
差出人:      山口XX <xxx@mapry.jp>
件名:        Re: 製品の不具合について
ヘッダ:      [全文]
本文抜粋:    「お前ら台湾人はみんなこうなのか」「金払わなければ断貨」
SHA256:      [計算済]
立証趣旨:    被告の差別的・脅迫的言辞、不法行為（民法 710）、脅迫罪（刑法 222）構成要件該当
```

#### G3. ソースコード TODO 未実装コメント 抜粋

**何のため**: **刑法 246 詐欺罪の故意の動かぬ証拠** + 民事 96 条詐欺主張

**現状**: defects.json D-31 に「コード内コメントに『TODO: 危険』『未実装』明記」記載あり

**作業**:
1. 該当ソースコードファイルを特定（既存分析資料に位置情報あるか確認）
2. TODO コメント該当行を**スクリーンショット**（行番号付き）
3. **git blame** 実行（誰がいつコミットしたか）
4. ソースコード入手経路の合法性確認（NDA は締結なしと劉氏言明 → 入手経路 OK）

**git blame コマンド**:
```bash
cd <mapry-source>
git blame -L 27,30 path/to/file.py  # 27 行目の TODO「沒辦法了就用這文言上吧」
```

**整理表**:
```
甲第11号証 — ソースコード故意立証資料
ファイル:    [path/to/file.py]
該当行:      第27行（TODO コメント）
コミット:    [hash] [author] [date]
スクリーンショット: [画像 PDF]
git blame:   [出力]
立証趣旨:    被告の製品欠陥の認識（刑法 246 故意立証の核心）
```

## 6. 注意点

### 6.1 chat.md（Gemini 対話）の扱い
- **訴訟証拠としては使わない**（AI 出力は信頼性なし）
- **配付資料にも含めない**（誤導の元）
- 個人参考用のみ、本書 + 弁護士判断で進める

### 6.2 機体現物の保全
- 墜落機体の物理証拠（D-34 バッテリーコネクタ焼損等）は**現物保全**を継続
- 機体所在地と保管環境の記録（温度、湿度、改変なし）
- 第三者鑑定時まで開封・分解しない

### 6.3 メール証拠の改竄防止
- 元の `.eml` ファイル**原本保持**
- SHA256 ハッシュ計算（G2）と日付スタンプ
- バックアップ複数箇所（USB、クラウド、CD-R）

### 6.4 訴訟費用の自己負担リスク
- 訴訟物価額 ¥6,800 万 → 印紙 ¥21 万
- 敗訴の場合は**訴訟費用の一部負担あり**
- 法テラス利用で着手金分割 + 敗訴時免除の可能性
