## 1. 刑法 246 条 詐欺罪（★★★ 最有力）

### 条文（pgvector statutes より）
> 人を欺いて財物を交付させた者は、十年以下の拘禁刑に処する。
> 前項の方法により、財産上不法の利益を得、又は他人にこれを得させた者も、同項と同様とする。

### 構成要件と本件証拠の対応

| 要件 | 本件における事実 | 証拠（甲号証） |
|---|---|---|
| ① 欺罔行為 | 「市售品（既製品）」「日本製完成品」として販売 → 実体は v0.0.0 開発版 + DJI Matrice OEM 移植 + ZED-F9P 機能未実装 | 注文書、契約書、Invoice、defects.json D-08/D-27/D-31/D-32 |
| ② 錯誤の惹起 | 劉氏が「完成品」と信じて契約 | 内容証明 2026-04-06、両者の往復メール |
| ③ 処分行為 | ¥6,510,000 の代金送金 | Invoice + 送金証明（顧客梁氏） |
| ④ 因果関係 | 「完成品」と信じなければ送金せず | （上記①②③で因果連鎖明白）|
| ⑤ **故意** | **ソースコード内に「TODO: 危険」「未実装」コメントを明示的に残しながら出荷** | **D-31（コードコメント現物）**、これは故意の直接証拠 |

### 関連判例（pgvector retrieval）

ソースコードを用いた故意立証の判例群（C-02 検索でヒット）:

- **平成30(ネ)10052** 知的財産高等裁判所 — ソースコードの「被告製品で意味を持たない記載」が認識証明として援用された
- **平成29(ネ)10007** 知的財産高等裁判所 — 「ソースコードを利用できないことを認識していた」立証
- **平成27(ワ)16423** 東京地方裁判所 — ソースコード鑑定による事実認定
- **平成25(ネ)10085** 知的財産高等裁判所 — 通知書 + 陳述書 + ソースコード分析の組合せ

これらは民事だが、**ソースコードからの認識（=故意）立証ロジック**は刑事告訴状でも有効に援用できる。

### 反論への備え

mapry 側が言いそうな反論と再反論:

| mapry の反論 | 想定再反論 |
|---|---|
| 「v0.0.0 はバージョニング規約に過ぎず、機能完成品である」 | config.json `0.0.0` ↔ config.yaml `1.0.0` 不整合（D-21）、git release 署名なし（D-08）、TODO コメント実装（D-31）等の総合性で反論 |
| 「販売代理店契約だから完成品の保証なし」 | 0818 契約書と Invoice の文言を精査、「市售品」性は商品引渡前のメール往復で立証 |
| 「劉氏は技術者でリスクを認識すべきだった」| 専門知識の不対称 + 内容証明での通知後も対応せず（重過失） |

### 告訴先

- **被告本店所在地警察署 生活安全課 or 知能犯係**
- 大型事件は**地検直告**（被害額 ¥6,800 万、社会的影響大）も検討

### 注意

刑事詐欺の起訴率は日本で**極めて低い**（民事の問題と認定されやすい）。但し:
1. 警察捜査 → mapry にとって重大圧力（社会的信用失墜）
2. 不起訴でも処分理由通知書を**民事訴訟で援用可能**
3. **検察審査会への申立**で再検討可能（特に被害金額が大きい場合）

## 2. 刑法 233 条 信用毀損・偽計業務妨害罪（★★★ 見落とし注意）

### 条文（pgvector statutes より）
> 虚偽の風説を流布し、又は偽計を用いて、人の信用を毀損し、又はその業務を妨害した者は、三年以下の拘禁刑又は五十万円以下の罰金に処する。

### 本件における当てはめ

これは **mapry → 劉氏に対する** 罪。「偽計」（=不正な計略・欺罔行為）を mapry が用い、劉氏の「業務」（=台湾林業ドローン市場開拓事業）が妨害された構図。

| 要件 | 本件における事実 |
|---|---|
| ① 偽計 | v0.0.0 開発版を完成品偽装、ZED-F9P 機能偽装、OEM 移植を独自設計と表記 |
| ② 業務 | 劉氏の台湾政府委託案件（新竹縣五峰鄉、正昌製材公司林地） + 高知大学院研究 |
| ③ 信用毀損 or 業務妨害 | **墜落事故 → 台湾政府・クライアントへの信用失墜（数千万円規模）** |
| ④ 故意 | （上記 246 条と同じ立証） |

### なぜ「見落とし注意」か

詐欺罪（246 条）は「mapry が劉氏から金を取った」面に注目するが、信用毀損・偽計業務妨害罪（233 条）は「**mapry の偽計で劉氏の事業が壊された**」面を捉える。**逸失利益請求の刑事面の裏付け**になる。

民事 ¥3,500 万逸失利益請求と刑事 233 条告訴が**同一の事実関係を別の法律構成で捉える**形になる → 民事の心証強化に直結。

要 retrieval 追加。本書付録の pgvector 検索を弁護士面談時に深掘り。

## 3. 刑法 222 条 脅迫罪（★★ 中）

### 条文
> 生命、身体、自由、名誉又は財産に対し害を加える旨を告知して人を脅迫した者は、二年以下の拘禁刑又は三十万円以下の罰金に処する。

### 当てはめ

| 要件 | 本件における事実 |
|---|---|
| ① 害悪告知 | 「金を払わなければ断貨」 = 劉氏の**営業（=財産）** への害悪告知 |
| ② 一般人の畏怖 | 代理店事業者が断貨されれば事業継続不能 → 畏怖該当 |
| ③ 故意 | メール現物にあり |

### 困難点

- mapry 側「正当な権利行使（売買代金未払債権の保全としての出荷停止）」と主張する可能性
- 反論: **代金は既に全額支払済（Invoice + 送金証明）** → 出荷停止に正当根拠なし → 純粋な脅迫
- 即ち「金を払わなければ」の「金」が**何の金か** を立証する必要。追加要求の金か、既払代金の追加か等。

### 証拠の優先確認事項

→ **6/15 面談までに**: 該当メールの**全文 + 受信日時 + ヘッダー** を確定。「金」の意味を文脈で確定。

## 4. 不正競争防止法 21 条（虚偽表示罪）（★★ 中〜高）

### 条文要旨

商品の品質、内容、製造方法、用途、数量について誤認させる表示を、不正の目的をもって行った者を罰する（5 年以下の懲役 or 500 万円以下の罰金）。

### 当てはめ

| 要件 | 本件における事実 |
|---|---|
| ① 商品 | Mapry M4-0 ドローン |
| ② 誤認表示 | 「市售品完成品」表記、ZED-F9P RTK 機能搭載と仕様書記載、独自設計表記（実は DJI OEM 移植） |
| ③ 不正の目的 | 高額販売（¥651 万） + SBIR 補助金獲得（要検証） |

### 告発先

- **警察**（刑事手続）
- **経済産業省**（行政指導）
- **公正取引委員会**（不公正な取引方法として）

景品表示法（消費者庁）は一般消費者向けが原則のため、B2B 中心の本件では不正競争防止法の方が筋がよい。

## 5. 不適用または困難なもの

### 5.1 刑法 104 条 証拠隠滅罪 — **不適用**（私の ACTION_PLAN の誤り訂正）

条文（pgvector statutes より）:
> **他人の**刑事事件に関する証拠を隠滅し、偽造し、若しくは変造し、又は偽造若しくは変造の証拠を使用した者は、三年以下の拘禁刑又は三十万円以下の罰金に処する。

→ 条文に「**他人の**」とある通り、**自己の刑事事件の証拠隠滅は本罪に該当しない**。
D-20 の `git reset --hard` 自動破壊は mapry **自社の証拠** であり、本罪では訴追不可能。

但し:
- **民事訴訟法 224 条「文書提出命令違反 + 証明妨害」** として、裁判所が**証拠不存在の事実を当事者の主張通り認定可能**
- 訴訟戦略上は十分強力

### 5.2 刑法 230 条 名誉毀損罪 / 231 条 侮辱罪 — 公然性要件で困難

条文:
- 230 条: 「**公然と**事実を摘示し、人の名誉を毀損…」
- 231 条: 「事実を摘示しなくても、**公然と**人を侮辱…」

→ 私的な往復メール（mapry 山口氏 → 劉氏）は**公然性なし**。第三者への送信や公開がなければ不該当。

### 代替策: 法務省人権擁護局への「人権救済申立」

法務省人権擁護局は「人権侵犯事件調査処理規程」に基づき、**公然性要件なし** で差別言辞を救済対象とする。

- 強制力は弱い（指導・啓発レベル）
- 但し**公的記録**として残り、民事訴訟の心証材料 + 慰謝料額算定の根拠に
- インターネット上で書かれた場合は、加害者へのプロバイダ責任制限法ベースの開示請求も別途可能

→ **6/15 弁護士に「人権救済申立の手続き + 訴訟との併用効果」を確認**

### 5.3 刑法 211 条 業務上過失致死傷罪 — 不該当

「業務上必要な注意を怠り、よって**人を死傷させた**者」の罪。本件は**部品高空落下のみで人身被害なし** → 構成要件不該当。

但し:
- **国土交通省航空局への墜落事故報告**（無人航空機制度）の材料
- 「あわや人死にだった」という事案の重大性として民事慰謝料・公的告発の文脈で言及可能

## 6. 三線（刑事・民事・行政）並行戦略

```
                         ┌──────────────────────────┐
                         │   弁護士相談（6/15 予定）  │
                         │   freelance110 + 民事代理  │
                         └──────────────────────────┘
                                       │
                ┌──────────────────────┼──────────────────────┐
                ▼                      ▼                      ▼
        ┌───────────────┐      ┌───────────────┐      ┌───────────────┐
        │  刑事告訴      │      │  民事訴訟     │      │  行政告発     │
        │                │      │                │      │                │
        │ ★ 刑法246 詐欺 │◀────▶│ 主位: PL3条   │◀────▶│ 経産省 SBIR    │
        │ ★ 刑法233 偽計 │ 同一 │      民415   │ 同一  │ 公取委 優越的  │
        │   業務妨害     │ 事実 │      民709   │ 事実  │   地位濫用     │
        │   刑法222 脅迫 │ で   │ 予備: 民96   │ で    │ 国交省 航空局  │
        │ 不正競争21条   │ 並進 │      民710   │ 並進  │ 法務省 人権局 │
        │                │      │                │      │   （差別救済） │
        └───────────────┘      └───────────────┘      └───────────────┘
                │                      │                      │
                ▼                      ▼                      ▼
        起訴・不起訴→            判決・和解→              行政指導・
        検察審査会               損害賠償・印紙            返還命令・公表
        （民事援用可）            訴訟物価額再検討
```

**並進の意義**:
- 3 つ全てが同一証拠群（甲号証セット）で動かせる → 1 つの証拠整理が 3 倍働く
- 一方の進捗が他方の交渉力になる（例: 警察捜査開始 → 民事和解応諾の圧力）
- mapry にとって「コスト 3 倍」の戦線拡大 → 早期和解動機
