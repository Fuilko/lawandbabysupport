## 第 2 章　プロジェクト概要（30 秒サマリー）

> **LegalShield は、社会的弱者（DV 被害者・ストーキング被害者・フリーランス被害者・外国籍研究者）が、
> 弁護士費用ゼロで「自己防衛」「証拠保全」「法的書面起案」を可能にする AI 駆動の本人訴訟支援プラットフォームです。**

技術スタック:
- **フロントエンド**：iOS Native (SwiftUI, iOS 17+)
- **バックエンド**：Python (FastAPI), DuckDB, LanceDB（ベクトル検索）
- **AI**：sentence-transformers 多言語埋め込み + LLM (Ollama Llama 3) on-device
- **OSS**：MIT License、フル公開

## 第 5 章　本日の Xcode 実演について

GitHub から最新版をクローン後、以下の手順で iPhone Simulator 上で起動できます。

```bash
# 1. クローン (任意の場所で)
git clone https://github.com/Fuilko/lawandbabysupport.git
cd lawandbabysupport/ios/LegalShield

# 2. XcodeGen（初回のみ）
brew install xcodegen

# 3. .xcodeproj 生成
xcodegen generate

# 4. Xcode で開く
open LegalShield.xcodeproj

# 5. Simulator (iPhone 15 Pro) を選択して ⌘R
```

実演内容:
1. **Evidence Vault** — 証拠写真の即時 SHA-256 ハッシュ生成
2. **防誘導取証 Copilot** — マイクで質問内容をキャプチャし、リアルタイムで法的助言を表示
3. **法律文書ジェネレーター** — Mapry 仲裁案件用「準備書面（第 1）」のリアル生成デモ
4. **Audit Log** — 全アクションが改ざん不可な形で記録される様子
