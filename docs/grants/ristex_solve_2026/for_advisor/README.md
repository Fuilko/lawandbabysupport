### B. PDF 化したい場合

**方法 1（推薦）**：DOCX を Word で開く → 「名前を付けて保存」→ PDF 形式

**方法 2**：HTML をブラウザで開く → Ctrl+P → 「PDF として保存」

**方法 3（自動）**：本フォルダで `.\build.ps1` 実行 → DOCX/HTML/PDF 一括生成（pandoc + MiKTeX 必要、すでにインストール済）

### C. 対面持参用
