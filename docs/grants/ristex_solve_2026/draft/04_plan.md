# 様式4-1　目標・実施計画

## 1. 目標・実施計画（A4 3 ページ厳守）

### 4-1.1 創出される価値と実施期間終了時点（2030 年 3 月）の達成目標

3 領域（性暴力・労働・消費者）× 3 地域（高知・東京・秋田）の実証研究を通じて、以下の達成目標を掲げる。これらは検証可能な指標として位置付け、外部の独立評価を受ける前提で運用する。

| 達成指標 | 検証方法（想定） |
|---------|--------------|
| 各実証地域における登録支援対象者の累計人数 | 連携 NPO の運用記録 |
| 暗号学的な共通性検知の累計件数 | 共通基盤の監査ログ |
| 支援機関による仲介・接続事例の累計件数 | NPO・弁護士会等への引継統計 |
| 当事者の開示請求・削除請求への対応率（適時性） | 外部監査 |
| 透明性レポートの発行回数 | 公開実績 |
| 自治体・支援機関との連携実績の数 | 協定書・MOU |
| 政策提言・立法府への報告・査読論文の本数 | 公開実績 |
| 他地域への展開可能性の検証地数 | 展開可能性レポート |
| プラットフォームのコード公開状況（OSS） | 公開リポジトリ |

### 4-1.2 全体アプローチ（バックキャスティング）

公募要領 p.12 の「目指すべき姿から立ち戻って計画を立てるバックキャスティング」に従い、2030 年ビジョンから 4 段階（A〜D）の実施計画に展開する。

**想定されるリスクと対応**

| 想定リスク | 対応 |
|-----------|------|
| プライバシー保護型照合における偽陽性 | 複数投影による多重判定、支援機関による仲介を必須とし、自動通知のみで完結しない設計 |
| 加害者側からの逆利用の懸念 | 端末側で生成された識別子のみを用い、サーバ側に直接の個人情報を持たない設計 |
| 自治体・支援機関との協力体制構築の遅延 | 採択後に段階的な対話を進め、内閣府・自治体の既存施策（自殺総合対策、孤独孤立対策等）との接続を優先 |
| 個情法・関連法令の改正への対応 | 法務監査体制を継続的に運用し、半年ごとに設計レビューを実施 |
| プロジェクト終了後の継続資金 | 自治体委託費・社会的インパクト投資・財団助成等、複数経路の調達計画を初年度に策定 |
| 採択評価における「技術が前面に出ている」との指摘の回避 | 全ての実証フェーズで協働実施者主導の運用とし、研究代表者は支援・倫理担保に徹する |

### 4-1.3 4 つの研究開発項目とマイルストーン

#### A. 基盤構築・倫理整備（2026.10 〜 2027.9、12 ヶ月）

| 中項目 | 担当（想定） | 内容 |
|-------|----------|------|
| A-1 倫理審査体制構築 | 研究代表者 G | 大学倫理委員会への申請、外部倫理委員（弁護士・当事者経験者・情報倫理学者）の招集 |
| A-2 鍵分散管理の運用設計 | 協働実施者 G・法務協力者 | 複数主体合意プロセスの策定（支援機関・法務専門家・運用主体・倫理委員会の協働形態） |
| A-3 共通基盤のプロトタイプ整備 | 研究代表者 G（情報セキュリティ） | OpenMined PSI、SQLCipher 等のオープンソース基盤の統合、第三者によるセキュリティ評価 |
| A-4 当事者・支援者対話 | 協働実施者 G | 当事者経験者・支援団体・専門職との要件定義対話 |

- **マイルストーン M1**：2027.3 までに倫理審査の承認取得
- **マイルストーン M2**：2027.6 までに要件定義書を公開
- **マイルストーン M3**：2027.9 までに共通基盤の第三者評価を完了

#### B. 三地域パイロット実証（2027.4 〜 2028.9、18 ヶ月）

| 中項目 | 担当（想定） | 内容 |
|-------|----------|------|
| B-1 高知パイロット（地域実証拠点） | 協働実施者 G・高知の連携機関 | 自伐型林業の分散思想を運用設計に反映、3 領域横断のパイロット |
| B-2 東京パイロット（首都圏連携拠点） | 連携 NPO・弁護士会 | 多分野 NPO 連携、政策接続の基点 |
| B-3 秋田パイロット（地域格差検証拠点） | 秋田大学等学術連携・地域 NPO | 自殺・孤立・医療空白等の地域特性を踏まえた検証 |
| B-4 領域横断機能の検証 | 全グループ | 法令 RAG・行政手続支援・暗号学的協調検知の 3 領域同時稼働 |

- **マイルストーン M4**：2027.12 までに最初の地域での運用開始
- **マイルストーン M5**：2028.6 までに 3 地域での運用実績の中間評価
- **マイルストーン M6**：2028.9 までに領域横断の運用報告を取りまとめ

#### C. 横展開と国際連携（2028.4 〜 2029.9、18 ヶ月）

| 中項目 | 担当（想定） | 内容 |
|-------|----------|------|
| C-1 国内他地域への展開可能性試験 | 全国の支援団体ネットワーク | 鍵分散方式による地域分散運用の試行 |
| C-2 台湾 LASS コミュニティとの対話 | 研究代表者 G・台湾連携 | オープンソース精神の共有、現代婦女基金會等との情報交換 |
| C-3 政策接続 | 全グループ | 内閣府・厚労省・関連自治体への中間報告、政策提言素案の作成 |
| C-4 学術成果の公開 | 研究代表者 G | 査読論文の投稿、国際会議発表 |

- **マイルストーン M7**：2029.3 までに国内他地域での試行運用 1 件以上
- **マイルストーン M8**：2029.6 までに台湾コミュニティとの対話実績を公開
- **マイルストーン M9**：2029.9 までに政策提言素案を公開

#### D. 事業計画策定・自立移管（2029.4 〜 2030.3、12 ヶ月）

| 中項目 | 担当（想定） | 内容 |
|-------|----------|------|
| D-1 継続運用体制の整備 | 協働実施者 G | 一般財団法人化等の運用主体の準備、定款・規約・収支計画の策定 |
| D-2 継続資金源の確保 | コーディネーター | 自治体委託費、社会的インパクト投資、財団助成等の調達 |
| D-3 ハンドブック・契約雛形の公開 | 全グループ | 他地域展開のための運用マニュアル、契約雛形、技術文書の公開 |
| D-4 最終成果報告 | 全グループ | 国内外関係者を招いた最終報告会 |

- **マイルストーン M10**：2029.12 までに運用主体の準備完了
- **マイルストーン M11**：2030.3 までに継続資金 3 年分の確保見込みを獲得
- **マイルストーン M12**：2030.3 までにハンドブック等の公開、最終報告会の開催

### 4-1.4 各実施項目の担当主体（概念整理）

| 項目 | 研究代表者 G | 協働実施者 G | コーディネーター | 解決策定着主体 | 他地域展開主体 |
|------|------|------|------|------|------|
| A 基盤 | ◎主導 | ○協力 | - | - | - |
| B 三地域 | ○支援 | ◎主導 | ◎調整 | 各地域連携機関 | - |
| C 横展開 | ○支援 | ◎主導 | ◎調整 | 同上 | ◎全国支援団体ネットワーク |
| D 事業計画 | ○支援 | ◎主導 | ◎調整 | 運用主体（法人化予定） | 同左 |

公募要領 Q&amp;A 「コーディネーター、解決を定着させる担い手、他地域への展開の担い手」を概念上明示する。具体的な担い手は採択後の対話を通じて確定する。

### 4-1.5 提案段階で不足する研究部門と連携計画

- **暗号応用研究**：採択後、国内大学および OpenMined Japan 等のコミュニティと連携を計画
- **法令 RAG／LLM 応用研究**：採択後、自然言語処理・法情報学の研究者と連携を計画
- **当事者参画研究の方法論**：採択後、参加型アクションリサーチの研究者と連携を計画
- **疫学的評価**：採択後、公衆衛生大学院と委託契約を計画
