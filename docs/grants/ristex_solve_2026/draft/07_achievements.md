- ［IRB 申請・倫理審査経験］
- ［プログラム総括・アドバイザーレベルとの対話経験／審議会委員経験］
- ［当事者参画型研究の経験（参加型アクションリサーチ等）］

# 様式6-2　研究・活動実績（協働実施者）（A4 3 ページ厳守）

> このセクションは協働実施者が確定次第、本人実績で全面差し替えてください。

## 1. 協働実施者：［氏名］

### (1) 研究者情報等
- 氏名：［氏名］
- 所属・役職：NPO 法人△△ 代表理事
- 所属法人 設立年：［年］
- 専門領域：性暴力／DV／ストーカー被害者支援、心理的応急処置、ジェンダー暴力対策

### (2) 研究・活動・マネジメント実績（10 件以内）

> **記載指針**：研究開発に限らず、実社会での課題解決に貢献した活動実績を中心に。各項目に「提案の遂行への重要性」を簡潔に付す。

1. **［NPO の主要事業名・期間・実績数］**：例「性暴力被害者ホットライン運営（〇〇年〜現在、累計相談 〇〇件、連携機関 〇〇）」
   → 本研究関連性：B 三地域パイロットの直接基盤
2. ［政府・自治体審議会委員等の経験］
   → 関連性：内閣府／自治体との協定締結の信頼基盤
3. ［MOU／協定締結実績（警察、弁護士会、自治体）］
   → 関連性：採択後の協定締結の実証
4. ［受託事業・助成事業の代表実績、規模、成果］
   → 関連性：本予算規模のマネジメント能力
5. ［論文・書籍・報告書（あれば。報告書ベースで構わない）］
   → 関連性：当事者視点の発信能力
6. ［国際会議発表／国際 NGO 連携実績］
   → 関連性：台湾連携の現場基盤
7. ［受賞・社会的評価］
   → 関連性：協働実施者としての適格性
8. ［メディア出演・政策提言活動］
   → 関連性：成果アウトリーチ能力
9. ［研修事業・人材育成実績］
   → 関連性：他地域展開のためのキャパシティビルディング
10. ［その他、提案の遂行能力を示す実績］

### 補足：マネジメント能力の説明

協働実施者は、以下の点で本研究提案を遂行するための能力と、社会課題解決の現場代表者としての適格性を有する：

- ［NPO 経営年数と組織規模］
- ［累積相談件数・支援件数］
- ［警察・自治体・弁護士会との連携実績］
- ［当事者参画型運営の経験］
- ［マルチステークホルダー調整経験（行政・学界・メディア・国際 NGO）］
- ［プロジェクト終了後の事業継続意思と具体的計画］
