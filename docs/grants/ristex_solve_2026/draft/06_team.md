# 様式5-1　実施体制（全体）

## 1. 実施体制（概念）

本研究の実施体制は、研究代表者グループ（学術・倫理側）と協働実施者グループ（現場・運用側）を二本柱とし、これに地域別および機能別の主たる実施者グループを配する構成を構想している。具体的な所属機関名・氏名は、採択後の対話を経て確定する。

| グループ | 想定する役割 |
|---------|----------|
| **研究代表者グループ（人文社会・倫理担当）** | 全体マネジメント、倫理審査、論文化、政策提言 |
| **研究代表者グループ（情報セキュリティ担当・主たる実施者 1）** | 共通基盤の設計、暗号応用、第三者監査窓口 |
| **協働実施者グループ** | 現場運営、当事者対話、運用主体の準備 |
| **連携グループ（高知）** | 西日本・地域実証拠点 |
| **連携グループ（東京）** | 首都圏・支援機関連携拠点 |
| **連携グループ（秋田）** | 東北・地域格差検証拠点 |
| **コーディネーターグループ** | 各機関間の調整、政策接続 |
| **海外連携（台湾 LASS コミュニティ等）** | 国際的な知見共有（JST との直接契約は希望しない） |

## 2. その他の協力者・機関（候補）

採択後に対話を進めることを想定している連携先カテゴリーを以下に示す。具体的な機関名・人物名は、本フェーズの初期段階で順次確定していく。

- 内閣府男女共同参画局、こども家庭庁、厚生労働省自殺総合対策推進センター等の関係省庁
- 都道府県警察生活安全部門、自治体担当部局
- 弁護士会（DV・性暴力・労働・消費者問題関係委員会）、法テラス
- 性暴力・DV・ストーカー領域の支援団体
- ハラスメント・労働被害領域の支援団体（外国人支援団体を含む）
- 消費者問題関連団体・消費生活センター
- 全国規模の支援団体ネットワーク
- 大学倫理委員会、外部倫理委員（弁護士・当事者経験者・情報倫理学者）
- 国内外のオープンソースコミュニティ（OpenMined Japan、Code for Japan 等）
- 台湾 LASS コミュニティ、現代婦女基金會、台北市立大學等

# 様式5-2　実施体制（研究代表者グループ）（A4 2 ページ厳守）

## 1. 研究代表者グループの構成（想定）

| 役割 | 概要 | エフォート目安 |
|------|------|----------|
| 研究代表者 | 大学等所属の研究者。情報法／ジェンダー研究／犯罪学／公衆衛生／情報セキュリティ等のいずれかの分野で本テーマに関心を持つ研究者 | 40% |
| 博士研究員 1 名 | 倫理審査・文献整理・論文化を担当 | 100% |
| RA 2 名 | データ分析、ステークホルダー対話の記録整理 | 各 50% |
| 主たる実施者 1（情報セキュリティ分野） | 共通基盤の設計と暗号応用を担当 | 25% |
| 暗号応用研究員 1 名 | OpenMined PSI 統合、運用支援 | 100% |
| 外部倫理委員 3 名 | 弁護士、当事者経験者、情報倫理学者 | 委嘱 |

## 2. 構想における位置付け

研究代表者グループは、本プロジェクトの **学術的・倫理的中核** として、以下を担う。

1. 情報法・社会科学・人文知を統合した学際的研究設計
2. 大学倫理委員会への申請と継続的倫理審査
3. プロトコルの理論的正当性検証と外部第三者によるセキュリティ評価の委託
4. プログラム総括・アドバイザーとの対話、他 SOLVE 採択プロジェクトとの相互学習

特に、公募要領 p.21 Q&amp;A で指摘される「**社会課題への取り組みよりも、技術シーズの研究開発が前面に出ている**」という典型的な不採択理由を回避するため、研究代表者グループの活動は、常に協働実施者グループの現場ニーズに応じる位置付けとする。

## 3. 特記事項

- 安全保障輸出管理：研究代表者所属機関の既存規程に準じて運用する。
- 異動予定がある場合は確定次第連絡する。
- 研究代表者・主たる実施者の最終的な氏名・所属は、書類選考通過後または採択後の手続きにおいて速やかに更新する。

# 様式5-3　実施体制（協働実施者グループ）（A4 2 ページ厳守）

## 1. 協働実施者グループの構成（想定）

| 役割 | 概要 | エフォート目安 |
|------|------|----------|
| 協働実施者 | 社会課題に取り組む団体・組織の代表者 | 35% |
| コーディネーター 1 名（採択後雇用） | 機関間調整、政策接続、事業計画素案 | 100% |
| 現場支援員 2 名 | 当事者対話、心理的応急処置、仲介対応 | 各 50% |
| 法務アドバイザー 2 名（委嘱） | 法務監査、弁護士会との連携 | 委嘱 |
| 当事者経験者顧問 3 名（委嘱） | 研究設計への当事者視点の統合 | 委嘱 |

## 2. 構想における位置付け

協働実施者グループは、本プロジェクトの **現場実装と事業継続性の中核** である。

1. 当事者の声を実証研究の各段階で集約・反映
2. NPO・自治体・弁護士会との連携を実務的に進める
3. 実証研究における倫理的運用
4. プロジェクト終了後の継続運用主体としての準備

公募要領 p.15「**協働実施者は社会課題に直面し研究者と協力して課題を解決することを希望する団体・組織の代表**」「**団体・組織を実質的に代表し社会課題に実際に取り組むことを想定**」に厳密に合致する。役職的な代表者である必要はないとされているが、本提案では実質的に代表する立場の方を協働実施者とすることを想定している。

## 3. 特記事項

- 採択後、運用主体としての法人化（一般財団法人または公益的運用主体）を見据えた組織体制強化を進める。
- 安全保障輸出管理：研究代表者所属機関の規程に準じて運用する。

## 主たる実施者 2〜4 グループ（地域別連携）

各地域における主たる実施者の所属機関・氏名は、採択後の対話を通じて確定する。以下に各地域における構成の想定を示す。

### 高知連携グループ（地域実証拠点）
- 高知大学関連機関、地域自治体、地域市民団体
- 自伐型林業実践者ネットワーク（地域知の協力）
- 地域に根差した支援者・専門職

### 東京連携グループ（首都圏連携拠点）
- 性暴力・DV 領域の支援団体
- ハラスメント・労働被害領域の支援団体
- 消費者問題関連団体
- 弁護士会の関連委員会
- 政策接続のための関係省庁との対話拠点

### 秋田連携グループ（地域格差検証拠点）
- 秋田大学医学部等の学術連携
- 自殺総合対策関連の地域団体
- 孤独孤立対策関連の地域団体
- 県内自治体との連携

---

## 海外連携グループ（台湾 LASS コミュニティ等）

JST との直接契約は希望しない。連携費用は協働実施者所属機関を経由して執行する。本フェーズでは海外での実証試験は行わず、可能性試験・実証試験は国内 3 地域に限定する（公募要領 Q&amp;A 遵守）。

安全保障輸出管理に関わる懸念がある場合は、研究代表者所属機関の規程に基づき個別審査を経る。
