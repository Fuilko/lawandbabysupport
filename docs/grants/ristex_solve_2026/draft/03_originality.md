# 様式3　提案の独創性・優位性（A4 2 ページ厳守）

## 3.1 国内外における関連分野の現状と動向

### 既存の被害者支援システムとその限界

| システム | 国・主体 | 機能 | 限界 |
|---------|--------|------|------|
| 110 番通報・専用通報アプリ | 日本（警察庁） | 緊急通報 | 単独通報が中心、複数当事者の協働的立件が困難 |
| DV 相談プラス | 日本（内閣府） | 電話・SNS 相談 | 加害者の重複性検知機能はない |
| 跟蹤騷擾防制法 e化通報 | 台湾（衛福部） | ストーカー被害通報 | 同上、当事者間の協働支援機能はない |
| Project Callisto | 米国（30 以上の大学等） | 暗号学的サバイバーマッチング | テキストのみで照合、改名等での回避が容易、対象が大学・IT 業界に限定 |
| 各種 SNS 上の #MeToo 運動 | 国際 | 当事者の発信支援 | プライバシー保護と名誉毀損リスクの両立が課題 |

### 関連分野の学術的動向

- **連続加害行為に関する研究**：Lisak &amp; Miller (2002) を起点に、米国 NSVRC「Rethinking Serial Perpetration」(2015) 等で繰り返し検証され、被害が偏在的に集中する構造が学術的に支持されている。
- **プライバシー保護型データ照合**：私的集合交差（PSI）の商用実装が成熟期に入り（Pinkas et al. 2018、OpenMined PSI、Microsoft APSI 等）、社会実装の準備が整いつつある。
- **市民参加型データと政策接続**：台湾 LASS／AirBox を題材とした Wong et al. *Nature Humanities &amp; Social Sciences Communications* (2022) が、市民データから政策提言につながる過程を学術的に検証している。
- **当事者プライバシーと安全の不可分性**：NNEDV「Privacy Matters」(2024)、UN Women Safety &amp; Security Toolkit 等が、被害者の安全とプライバシーの保護が不可分であることを国際的に確立している。
- **AI／LLM の法律支援応用**：海外では Stanford CodeX、欧州 Legal Tech 関連で進展、国内でも法律相談チャットボットの研究が増えている。

## 3.2 受益者から見た本提案の優位性

**「技術シーズの独自性ではなく、受益者から見た効果の観点」**（公募要領 p.16 様式 3 注釈）から記述する。

### 受益者の体験における変化（Before／After）

| 局面 | 現状（Before） | 本提案の実装後（After） |
|------|------------|--------------------|
| 被害・困難の直後 | 一人で抱え、相談先を探す段階で諦めることが多い | 端末上のセキュアな記録機能で自己防衛のための記録を残せ、心理的な選択肢が増える |
| 相談判断 | 一人での通報・相談を躊躇 | 「同じような状況の方が他にもいるか」を、自身のプライバシーを守ったまま確認できる |
| 共通性の発見 | 仕組みが存在しない | 暗号学的に共通性が確認されたときのみ、支援機関を介した安全な仲介が始まる |
| 連名告訴・集団的解決 | 制度上は可能だが運用上ほぼ実例なし | 支援機関による仲介で、本人の意思に基づく協働的解決の選択肢が開ける |
| プライバシー保護 | 異機関間で意図せず情報共有される懸念 | 顔・位置等の情報を外部サーバに送らずに済む設計 |
| 行政手続き | 書類作成の負担で諦める方が多い | 法令検索と書類作成補助で、申請・不服申立て等の選択肢が広がる |
| 心理的回復 | 孤立しやすい | 同じ経験を持つ仲間と安全につながる選択肢が生まれる |

### 国内外における独自性

1. **AI 法律支援と暗号学的協調検知の統合**：海外でも別々には存在するが、両者を**全法律領域に横断適用可能な共通基盤**として統合する取り組みは先進的である。
2. **多者合意型の鍵管理**：支援機関・法務専門家・運用主体・倫理委員会等の複数主体の合意なしには機微情報を復号できない設計により、単独主体への信頼集中を構造的に避ける。
3. **日台二拠点からの東アジアモデル**：「課題先進国・日本」モデル（公募要領 p.6）を、台湾 LASS コミュニティの理念と接続することで、東アジア地域の市民データ駆動型政策モデルの先行事例となる可能性がある。
4. **暗数定量化データベース（JUDB）との接続**：警察庁・内閣府・厚労省・衛福部等の公開統計と本研究の集約データ（一定人数以上の匿名集約のみ）を統合し、地域別の指標可視化を実現する（既に研究基盤として構築済）。

## 3.3 競合・先行研究との位置付け

| 先行事例 | 強み | 本研究との関係 |
|---------|------|--------------|
| Project Callisto（米国） | 10 年運用実績、査読論文（ACM COMPASS 2018） | 本研究はその発展形として、対象領域を拡張し、日本法制度との適合を図る |
| LASS／AirBox（台湾） | 市民データ駆動型政策の実証、Nature 査読 | 本研究は同コミュニティの理念を継承し、扱う対象を社会安全領域に拡張する |
| #MeToo 運動 | 国際的可視化と社会的合意形成 | 本研究は、声を上げる方々のプライバシー保護と被害の証拠保全を補完する |
| 既存の警察・行政の通報制度 | 公的な受付窓口 | 本研究は、これらの公的制度を補完し、複数当事者の協働的解決を可能にする予備的な仕組みとして機能する |

## 3.4 研究代表者・協働実施者の準備状況

本提案チームは、公募開始前から本研究の基盤となる以下を独自に整備しており、シナリオ創出フェーズに相当する準備が完了し、ソリューション創出フェーズに進める段階にある。

1. **JUDB（Japan Underreporting Database）**：日台の公開統計を集約する DuckDB ベースの分析基盤。47 都道府県マスター、複数の指標ビューを実装済（`legalshield/db/judb.py`、`judb_schema.sql`）。
2. **エビデンス基盤**：35 件の一次資料を `docs/refs/refs.bib` に整備済（内閣府男女間暴力調査原典、警察庁ストーカー・DV 年次報告、衛福部統計、Callisto、Lisak、LASS／AirBox 等）。
3. **法制度マッピング**：日本の関連法令（個人情報保護法、ストーカー規制法、DV 防止法、児童虐待防止法、消費者契約法、特定商取引法、労働法等）と台湾の関連法令（個資法、跟蹤騷擾防制法、家庭暴力防治法、性侵害防治法等）の条文と関連判例を整理済。
4. **クローラ群**：警察庁公開資料の自動抽出、台湾司法院判決書 API 連携を実装済。
5. **設計書群**：システム設計、安全性分析、利用規約案、運用設計を文書化済（`SAFETY_GIS_DESIGN`、`FEWN_DESIGN`、`EVIDENCE_BASE`、`EXPANSION_AND_DISPATCH_DESIGN`、`PIPELINE_AND_RISK_ANALYSIS`、`USER_AGREEMENT` 等）。
6. **既存技術シーズの統合プロトタイプ**：OpenMined PSI および顔特徴量抽出のオープン実装について、研究室レベルの動作確認を完了済。

これらの基盤により、本フェーズに採択された場合、初年度の早期から実証フィールドの構築段階に円滑に移行可能な状態にある。
