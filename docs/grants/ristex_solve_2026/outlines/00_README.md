## 共通方針

1. **二重受給回避**：各助成で経費区分を分離（人件費別人件、システム別モジュール、地域別実証）
2. **コア資料 1 セット流用**：紹介書 PDF / 35 件文献 / JUDB 可視化 / 予算テンプレ
3. **大学要件で振分**：要件あり = 研究代表者ベース（三菱）、要件なし = NPO ベース（日本・損保・AWS）
4. **採択順効果**：RISTEX or トヨタ採択 → 残り全助成の信用度上昇
