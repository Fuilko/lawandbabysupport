AI駆動型法律自助プラットフォームの構築と実証研究

―被害者の権利回復と司法アクセス格差の解消を目指して―
代表：劉建志（りゅう・けんじ）｜ 光伊フォレスト株式会社 / 高知大学大学院
トヨタ財団 共同研究助成（2027年度募集）｜ 2026年5月14日
1. 背景と課題

日本では年間約100万人が犯罪被害に遭うが（警察庁統計）、実際の被害実態はその2〜5倍と推計される（法務省「犯罪被害実態（暗数）調査」）。多くの被害者は「誰に相談すればいいかわからない」「弁護士費用が高い」「警察が被害届を受理してくれない」という三重の壁に阻まれ、権利を放棄している。

核心データ：民事訴訟における本人訴訟の勝率は32.4%。つまり、弁護士をつけない被害者の3人に1人しか勝てない。司法アクセスの格差は明確な数値として存在する。

加えて、近年顕在化した製品安全被害（ドローン事故、自動車・家電の欠陥等）では、個人が企業の隠蔽体制と巨大な法的リソースに対峙し、構造的な不利に晒される。被害者が孤立無援のまま、権利を諦める社会構造が固定化している。

2. 解決策：「LegalShield」プラットフォーム

本研究は、AI（大規模言語モデル×軽量言語モデルのハイブリッドアーキテクチャ）と政府統計・判例データを統合した「法律自助プラットフォーム」を構築する。従来の法テラス等の制度支援とは異なり、以下の3つの独自機能を実装する：

  • ① 被害届不受理防止モジュール：警察への届出・相談時の対話記録・証拠保全をAIガイド。行政不服申立・監査請求までのSOPを自動生成。
  • ② 逆転工学実証ベース：企業の製品欠陥を隠蔽する構造を、57GBのフォレンジック解析（SDカイメージ完全解析、ROS2コードレビュー、MAVLinkログ検証）により可視化する手法を体系化。
  • ③ プライバシー保証型AI：SLM（Small Language Model、Ollamaローカル運用）とLLM（大規模モデル）を状況に応じて振り分ける階層的推論アーキテクチャ。個人情報は端末内で処理し、外部への流出を原則禁止。
3. 実証手法

本研究は単なるシステム開発ではなく、「社会実装を伴う実証研究」として位置づける。2年間の助成期間中、以下の実証サイクルを回す：

フェーズ期間内容
Phase 1第1〜4ヶ月プラットフォームアルファ版開発。犯罪被害者コミュニティ（NPO、SNSグループ）との共創設計
Phase 2第5〜12ヶ月パイロット公開（β版）。ユーザーの問い合わせ・報告データを収集・匿名化分析
Phase 3第13〜20ヶ月効果検証。使用前后の「法的行動率」「専門家相談率」「被害届受理率」を比較
Phase 4第21〜24ヶ月学術論文執筆＋政策提言書作成。法務省・消費者庁への提言

独創的実証リソース：代表者自身が製品安全被害（ドローン墜落事故、被害額約7,000万円相当）を受け、57GBのフォレンジック証拠を収集・解析した実体験を持つ。この「被害者兼研究者」という二重の視座は、従来の法社会学研究には存在しない独自の実証基盤となる。

4. 期待される成果と社会変革

学術的成果：

  • 国際学術誌論文 1〜2編（犯罪学・情報学領域）
  • 学会発表 2〜3件（日本犯罪学会、情報処理学会等）
  • オープンソース公開（GitHub）

社会的インパクト：

  • 「被害届不受理」実態の可視化データ提供
  • 製品安全被害における企業隠蔽構造の透明化
  • 法テラス等既存制度との連携マップ構築

本研究が最終的に目指すのは、「デジタル技術によって、被害者が一人で戦わなくて済む社会」である。AIは弁護士の代わりにはならない。しかし、被害者が「知らない」「高い」「怖い」という理由で権利を放棄する現状に、科技の力で抵抗する手段を与えることができる。
