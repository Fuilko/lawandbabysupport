第1　告訴の趣旨

被告訴人の下記行為は、刑法第246条第1項（詐欺罪）に該当すると思料されるので、捜査の上、厳重に処罰されたく告訴いたします。

第2　告訴事実

被告訴人は、令和7年（2025年）12月24日頃、告訴人に対し、林業用無人航空機「M4-0」（機体番号: M4-00001、以下「本件機体」という）を代金800万円で販売するに際し、以下の欺罔行為を行った。

1. 開発版を完成品と偽った虚偽表示

本件機体のソフトウェアバージョンは「0.0.0」であり、社内的にも開発段階の試作機であった。にもかかわらず、被告訴人は「完成品」として保証書（甲第2号証）を発行し、見積書・注文書（甲第3号証）にも「開発版」である旨の記載を一切せず、告訴人に完成品であると誤信させた。

2. 重大な欠陥を認識しながら告知しなかった不利益事実の秘匿

本件機体の SD カードに記録されたフライトログのフォレンジック分析（甲第1号証）により、出荷前の段階で以下の異常が記録されていたことが判明した:

・磁力計異常（compass variance）: 4,509回

・システムリブート: 54回（出荷6日前の12月18日だけで25回）

・PreArm 警告（飛行不適合警告）: 2,781回

被告訴人はこれらの異常を認識し得る立場にありながら、告訴人に一切告知せず販売した。

3. バッテリー保護機能の意図的無効化

本件機体のパラメータファイル（甲第16号証）の分析により、以下のバッテリー安全保護機能が全て意図的に無効化（= 0 に設定）されていたことが判明した:

・BATT_LOW_VOLT = 0（低電圧警告: 無効）

・BATT_CRT_VOLT = 0（緊急着陸: 無効）

・BATT_FS_LOW_ACT = 0（低電圧フェイルセーフ: 無効）

これは製造者として安全基準を意図的に回避する行為であり、飛行中のバッテリー切れによる墜落の危険を認識しながら、利益のために出荷したことを示す。

4. 上記欺罔行為の結果

告訴人は本件機体が完成品であり安全に飛行可能であると誤信し、代金800万円を被告訴人に支払った（甲第＿号証: 送金記録）。告訴人が上記欠陥を事前に知っていれば、本件機体を購入することは絶対になかった。

5. 結果（損害の発生）

令和8年（2026年）2月3日、台湾において被告訴人の社員・宮田氏自身が操縦中に本件機体は墜落し全損した。墜落地点の近傍に告訴人がおり、80メートルの高度から落下した機体が告訴人に衝突する危険があった。

第3　被告訴人の故意を示す事実

以下の事実は、被告訴人が欠陥を認識しながら意図的に販売を推進した（未必の故意を含む）ことを示す:

（1）出荷6日前（12月18日）に25回のシステムリブートが記録されており、この時点で機体の不安定性を認識していた。

（2）出荷14日前（12月10日）、被告訴人代表・山口は告訴人に対し、PM契約の承認、鴻海・台塑との産業連携の承認、EV車両・ロボットへの展開希望を表明（甲第5号証）。台湾市場の巨大さを十分に認識しながら、14日後に欠陥品を出荷した。

（3）バッテリー保護の無効化は、パラメータを手動で「0」に設定する必要があり、偶然や初期値ではなく、意図的な操作である。

（4）バージョン「0.0.0」は業界標準において「開発初期版」を意味し、これに保証書を発行することは通常あり得ない。組織的な判断として虚偽表示が行われた。

（5）事故後（2月7日）、被告訴人は告訴人に対し「成果物がないので報酬は払えない」と回答（甲第＿号証）。16ヶ月間の業務投入を全て認識しながら、欠陥品による事故の責任を告訴人に転嫁しようとした。

第5　処罰を求める意思

告訴人は、被告訴人の上記行為について厳重な処罰を求めます。

なお、本件は単なる商取引上の紛争ではなく、以下の公益的観点からも捜査の必要性が高いと考えます:

（1）本件機体は80メートルの高度から墜落しており、人命に関わる危険があった。

（2）被告訴人は農林水産省 SBIR 補助金（約7.7億円）の受給企業であり、公金で開発した製品の品質管理体制に重大な疑義がある。

（3）同種の機体が他にも販売されている可能性があり、同様の事故が発生するおそれがある。
