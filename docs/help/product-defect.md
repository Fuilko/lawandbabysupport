# 製品の欠陥で事故・損害が発生したら何をすべきか

> 🛠️ **製品の欠陥が原因で**怪我をした／壊れた／お金を失った 場合、**製造業者の故意・過失を立証する必要はありません**（PL 法 3 条 — 無過失責任）。
> 「欠陥」と「損害との因果関係」を示せれば、損害賠償請求が成立します。

---

## 結論を先に：4 つのステップ

1. **証拠を保全する**（製品本体・破片・取扱説明書・写真・動画・購入記録）
2. **NITE と消費者庁に事故報告**（再発防止と他の被害者特定のため）
3. **188（消費者ホットライン）→ 消費生活センター**で事業者あっせんを試す
4. **解決しなければ弁護士会 → 仲裁センター（ADR）→ 訴訟**

---

## 緊急度別フローチャート

### 🚨 製品で怪我をした／火災・墜落・破裂など重大事故

| アクション | 連絡先 | 補足 |
|---|---|---|
| 救護・救急 | **119** | 怪我は最優先 |
| 製品事故報告 | **NITE 製品安全センター**（消費者庁経由 188） | リコール検討の起点 |
| 事故記録 | スマホで写真・動画 | 破片・焦げ跡・現場をすべて |
| 病院 | 診断書を必ずもらう | **「{製品名}による事故」と医師に伝える** |

### 🟡 製品が動作不能／重大な性能問題

| アクション | 連絡先 | 補足 |
|---|---|---|
| 1 次相談 | **188** | 消費生活センターでクーリングオフ・契約取消の助言 |
| 業界 ADR | NPC 等の業界 ADR | 自動車・家電・住宅は業界別 ADR あり |
| 弁護士相談 | 弁護士会 | PL 法・契約法に明るい弁護士 |

### 🟢 損害額が大きい／同種被害者がいる

| アクション | 連絡先 | 補足 |
|---|---|---|
| 弁護士会 ADR（仲裁センター） | 全国 35 か所 | 民事訴訟より早く・安く |
| 民事訴訟 | 簡易裁判所（140 万円以下）／地方裁判所 | 本人訴訟も可（少額訴訟は 60 万円以下） |
| 公共訴訟 | **CALL4**（公共訴訟プラットフォーム） | クラウドファンディング・社会化に強い |

## PL 法（製造物責任法）の基本

### 何が請求できるか

| 損害 | 例 |
|---|---|
| 治療費・入院費 | 怪我の治療 |
| 休業損害 | 仕事を休んだ分の収入減 |
| 慰謝料 | 精神的苦痛 |
| 物的損害 | 製品自体の代金、他の財産の毀損 |
| 逸失利益 | 後遺障害による将来収入の減少 |

### 立証責任

被害者が立証すべきは **「欠陥」と「損害との因果関係」だけ**。製造業者の故意・過失は不要（PL 法 3 条）。

「欠陥」とは：
1. **設計上の欠陥**：そもそも設計が安全でない
2. **製造上の欠陥**：一部の個体が不良品
3. **指示・警告上の欠陥**：取扱説明書の警告が不十分

### 時効

**事故から 3 年**または**製品の引渡しから 10 年**（PL 法 5 条）。
事故時点ですぐに弁護士相談を開始するのが望ましい。

## 証拠保全の優先度

> 🔴 **最重要**：製品本体を **絶対に廃棄しない**。事業者から「点検のために回収させてください」と言われても、応じる前に弁護士または消費生活センターに相談。

| 優先度 | 証拠 | 保存方法 |
|---|---|---|
| 🔴 最重要 | 製品本体・破片 | 段ボール・密封袋。**回収させない** |
| 🔴 最重要 | 事故時の写真・動画 | クラウド + ローカル両方 |
| 🟡 重要 | 取扱説明書・保証書 | 原本を保管 |
| 🟡 重要 | 購入記録（レシート・カード明細・メール） | スクショ + 原本 |
| 🟢 補強 | 同種事故の報道・SNS 投稿 | URL とスクショ |
| 🟢 補強 | NITE 事故情報データベース | 検索結果スクショ |

## 「言い方」テンプレート

**消費生活センター（188）に電話するとき**：
> 「○月○日に購入した {製品名} で、{事故概要} が発生し、{損害} を被りました。事業者対応がなされていないので、あっせんをお願いしたいです。」

**メーカーへの請求書面（弁護士同伴前）**：
> 「製造物責任法 3 条に基づき、貴社製造の {製品名} の欠陥により生じた損害について、書面でご回答ください。期限は本書面到達後 14 日以内とします。」

**弁護士会 ADR（仲裁センター）に申し立てるとき**：
> 「{製品名} の欠陥による損害について、相手方との交渉が決裂したため、仲裁手続きを申し立てたい。」

## 公共訴訟という選択肢

同じ製品の被害者が複数いて、**業界全体・規制官庁を動かしたい**場合、**公共訴訟**（public interest litigation）が有効です。

- **CALL4**（callfour.jp）：日本の公共訴訟プラットフォーム。クラウドファンディングで原告団形成・訴訟費用調達。
- 訴訟と同時に **メディア・SNS で社会化** することで、リコール・規制改正・業界慣行の変更を促す。
- 弁護士の選び方：PL 法・消費者法に加え、**メディア対応経験**のある弁護士。

## 産業別の専門 ADR

| 業界 | ADR 機関 |
|---|---|
| 自動車 | 自動車製造物責任相談センター（自動車 PL センター） |
| 家電 | 家電製品 PL センター |
| 住宅 | 住宅紛争処理支援センター |
| 医療機器 | 医療 ADR（弁護士会） |
| 化粧品 | 日本化粧品 PL センター |
| 玩具 | 日本玩具 PL センター |
| 食品 | 食品 PL 相談センター |

業界 ADR は **弁護士会 ADR より早く・安く**、専門知識も豊富。一方で「業界寄り」になりがちな批判もあるため、消費生活センターと並行で進めるのが良い。

## よくある失敗パターン

| 失敗 | 推奨 |
|---|---|
| 製品をメーカーに渡してしまう | **絶対に渡さない**。第三者調査機関へ |
| 「保証期間外」と言われて諦める | **PL 法は保証とは別**。10 年間有効 |
| 個人で交渉してこじれる | 早めに弁護士・ADR を挟む |
| 損害賠償額を低く見積もる | 慰謝料・逸失利益も含めて算定 |
| 証拠を整理しない | 時系列・スクショ・診断書を **一つの PDF にまとめる** |
