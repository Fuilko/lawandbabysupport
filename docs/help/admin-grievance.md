# 行政から理不尽な扱いを受けたとき、どう動くか

> 🏛️ **行政処分には「理由付記」と「不服申立教示」が義務**（行政手続法 8 条、行政不服審査法）。
> 書面で処分を受けたら、まず**「理由」と「不服申立期間（通常 3 ヶ月以内）」**を確認してください。

---

## このページが対象とする状況

- 児相による機械的な親子分離・面会拒否
- 生活保護の不当な却下・打ち切り
- 入管による在留不許可・退去強制
- 介護・障害福祉サービスの利用却下
- 公務員からの差別的・侮辱的対応
- 公文書改ざん・隠蔽の被害
- 学校・教育委員会のいじめ対応不適切

---

## 結論を先に：5 つのステップ

1. **書面化を要求する**（口頭の処分でも理由書を文書で出してもらう）
2. **証拠を残す**（メモ・録音・録画 — 公務員相手は **録音は適法**）
3. **法務省 みんなの人権 110 番（0570-003-110）に相談**（人権侵犯の調査が入る可能性）
4. **行政不服審査請求**（処分から 3 ヶ月以内）
5. **行政訴訟・国家賠償請求**（弁護士同伴）

---

## 緊急度別フローチャート

| アクション | 連絡先 | 補足 |
|---|---|---|
| 通報 | **110** | 公務員による暴行は刑法犯 |

| アクション | 期限 | 補足 |
|---|---|---|
| 行政不服審査請求 | **処分を知った日から 3 ヶ月以内** | 弁護士なしでも可能 |
| 異議申立 | 同上 | 処分庁宛 |
| 開示請求 | いつでも | 行政文書・個人情報の開示 |

## 「書面化を要求する」の重要性

口頭で「無理です」「該当しません」と言われた段階で帰ってはいけません。

**最初のお願い**：
> 「却下／拒否の理由を文書で出していただけますか？ 行政手続法 8 条に基づきお願いします。」

ほとんどの公務員はこの一言で態度が変わります。書面で出せない＝法的根拠が弱い可能性があります。

---

## 録音について

**公務員との対話を録音することは適法**です（プライバシー権より公務の透明性が優先される判例）。

- **「録音させていただきます」と一言伝える**のがマナーですが、伝えなくても証拠能力はあります（最判昭和 56 年 11 月 20 日）。
- 録音禁止と書かれた窓口でも、**自分の発言を録音する**ことは制限できません。

## 行政不服審査請求の書き方（テンプレ）

```
                  審査請求書

審査庁　○○県知事 殿（または ○○大臣 殿）

                  令和○年○月○日
                  審査請求人 氏名 ○○○○
                            住所 ○○県○○市…

一  処分庁
        ○○市福祉事務所

二  審査請求に係る処分
        令和○年○月○日付け ○○通知（処分番号 ○○）

三  審査請求に係る処分があったことを知った日
        令和○年○月○日

四  審査請求の趣旨
        上記処分の取消しを求める。

五  審査請求の理由
        1. 事実関係
        2. 法令違反の指摘（具体的条文を引用）
        3. 処分の不当性

六  処分庁の教示の有無及び内容
        有 / 無（教示書面の写しを添付）

七  添付書類
        処分通知書写し
        証拠書類一覧
```

**期間**：処分から 3 ヶ月以内。**1 日でも過ぎると却下**されます。

## 国家賠償請求とは

公務員の **違法な公権力行使**で損害を受けた場合、**国・自治体に対して損害賠償**を求める制度（国家賠償法 1 条）。

- 個別の公務員ではなく **国・自治体が被告**
- 時効：損害発生から **3 年**（民法 724 条）
- 弁護士費用：法テラス民事法律扶助で立替可
- 過去の認容例：
  - 児相の不適切な一時保護
  - 警察の過剰な職務質問
  - 学校・教育委員会のいじめ対応不作為
  - 入管の長期収容

---

## 公共訴訟（同種被害の社会化）

同じパターンの行政被害が複数あるなら、**公共訴訟**による構造的改革が可能。

- **CALL4**：原告団形成、クラウドファンディング、メディア露出
- 過去事例：技能実習生制度、夫婦別姓、ハンセン病、優生保護法
- 弁護士の選び方：公共訴訟経験・メディア対応・粘り強さ

## 当事者が陥りやすい罠

| 失敗 | 推奨 |
|---|---|
| 「お役所には逆らえない」と諦める | **書面化を求める**だけで状況が変わる |
| 期限を逃す | 不服申立は **3 ヶ月以内**。すぐ弁護士相談 |
| 口頭で議論を続ける | **書面・録音で証拠を残す** |
| 一人で抱え込む | **NPO・人権 110 番・弁護士会** の支援を組み合わせる |
| 感情的になる | **時系列メモ**を作って事実ベースで主張する |
