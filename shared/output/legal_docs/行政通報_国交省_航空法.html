2. 事故の概要

発生日時: 令和8年（2026年）2月3日　午前＿時＿分頃

発生場所: 台湾＿＿＿＿＿＿＿＿＿＿（座標: ＿＿）

機体: マプリィ M4-0（機体番号: M4-00001）

製造者: マプリィ株式会社

操縦者: 宮田＿＿（マプリィ株式会社 社員）

事故内容: 飛行中に制御不能となり、高度約80メートルから墜落。機体全損。付近に人員あり（告訴人含む）、人的被害は偶然なし。

3. 機体の安全上の問題

事故後の SD カードフォレンジック分析により、以下の事実が判明:

（1）出荷前から磁力計異常（compass variance）が 4,509 回記録

（2）出荷前にシステムリブートが 54 回発生（出荷6日前に25回）

（3）PreArm 警告（飛行不適合）が 2,781 回記録

（4）バッテリー保護機能が全て意図的に無効化（パラメータ = 0）

（5）ソフトウェアバージョンが「0.0.0」（開発版）のまま出荷

（6）上記にもかかわらず「完成品」として保証書付きで販売（代金 ¥800万）

4. 報告の趣旨

本件は、製造者が重大な安全欠陥を認識しながら出荷した疑いが極めて強く、同種機体が他にも販売・運用されている可能性があります。航空法第132条の85に基づく報告義務を果たすとともに、同種事故の防止のため、貴局による安全調査を要請いたします。
