## 第1ラウンド：ADR 初回期日

### 🔵 原告（あなた）の主張

```
1. 契約不適合 (民法§562)
   「version 0.0.0、user=m4dev01、開発用ファームウェアのまま納品。
    これは契約で約束された『林業測量用完成品』ではない。」

2. 製造物責任 (PL 法§3)
   「31 項目の安全欠陥。うち致命的欠陥:
    - 全バッテリー保護を意図的に無効化 (BATT_CRT_VOLT=0)
    - 磁力計上限 875 の 5.2 倍でもアーミング可能
    - 障害物回避センサー搭載なのに OA_TYPE=0（無効）
    - 出荷前 54 回リブート（SD カード bash_history で証明）」

3. 詐欺的行為 (民法§96)
   「出荷 6 日前の 12/18 に 25 回リブート → 重大な不安定を認識。
    同時期にPM契約・鴻海連携を承認 → 市場の巨大さを知りながら
    欠陥品を出荷。不利益事実の不告知。」

4. 損害
   - 機体代金: ¥800 万
   - 車両損傷: ¥XX 万
   - 逸失利益: SOP 30% × 台湾市場予想 + PM 報酬
   - 人件費: 8 ヶ月分の投入労力
```

### 🔴 被告（Mapry 弁護士）の予測される反論

#### 反論①: 「開発版」は合意の上

```
被告主張:
  「原告は販売代理店であり、製品の開発段階にあることを承知で契約した。
   version 0.0.0 はベータテスト段階であり、原告も了解していたはず。
   販売代理店契約は完成品の売買契約ではなく、共同開発の側面がある。」

★ あなたの反撃:
  「①契約書のどこにも『開発版』『ベータ版』の記載がない
   ②注文書に記載された金額は ¥800 万 — テスト品にこの金額は異常
   ③請求書 + 保証書が発行されている — 保証書がある = 完成品として販売
   ④山口自身が税関スペックを提供した = 正式輸出品として扱った
   ⑤NCC 認証の話をしていた = 市販を前提にしていた」

  → 証拠: 甲9号証の6（注文書）、請求書及び保証書、12/13 メール
  → 判定: この反論は崩壊する。保証書の存在が決定的。
```

#### 反論②: 過失相殺（原告にも過失がある）

```
被告主張:
  「2/3 の墜落は操縦ミスが原因。84 件の PreArm 警告を
   無視して飛行したのは原告（または現場担当者）の過失。
   高知のような強風環境での運用は仕様外。」

★ あなたの反撃:
  「①操縦者は Mapry 社員の宮田氏。原告は操縦していない。
   ②Mapry 自身の社員が操縦して墜落 = 製造者自身が使えない製品。
   ③PreArm 警告はパラメータで無効化されていた（ARMING_CHECK 改竄）
     → 警告が出ても飛べる設定にしたのは Mapry 自身。
   ④製品は『山岳林業用』として販売 → 山地での使用は仕様内。」

  → 証拠: 2/3 墜落動画（宮田氏操縦）、パラメータファイル
  → 判定: Mapry 社員が操縦して壊れた以上、過失相殺はほぼ0%。
          これは被告にとって致命的。
```

#### 反論③: 因果関係の否定

```
被告主張:
  「欠陥と墜落の因果関係が不明確。
   磁力計異常があったとしても、それが直接の墜落原因とは限らない。
   台湾の地磁気環境が原因の可能性もある。」

★ あなたの反撃:
  「①出荷前 syslog に 4,509 回の磁力計異常 → 台湾に来る前から異常
   ②日本でのテスト中にも Battery Failsafe 発動 (12/10 syslog.1)
   ③そもそも磁力計チェックを無効化した設定は Mapry が行った
   ④version 0.0.0 = 開発版のまま → 全体的な品質管理の欠如
   ⑤仮に磁力計が直接原因でなくても、31 項の欠陥のいずれかが原因。
     PL 法では『どの欠陥が原因か』の特定は不要。
     『欠陥があった + 損害が生じた』で足りる（PL法§3の立証構造）。」

  → 証拠: syslog 全文、パラメータファイル、フォレンジック報告
  → 判定: PL法の立証構造上、個別因果関係の特定は不要。被告の反論は空振り。
```

#### 反論④: 損害額の否認

```
被告主張:
  「¥800 万の返金は認めるとしても、逸失利益は過大。
   台湾市場の販売予測は単なる希望であり、確実な利益ではない。
   鴻海・台塑との連携も具体的な契約には至っていない。
   SOP 30% の手数料も実績がないため算定不能。」

★ あなたの反撃:
  「①正昌からの Order Form（実際の受注 = 実績）
   ②正昌からの送金証明（実際に代金が動いている）
   ③山口が 12/10 に PM 契約 + 鴻海/台塑連携を承認（メール証拠）
     → 被告自身が市場の実現性を認めていた
   ④EV・ロボット・トラックへの展開も山口が要望（12/10 返信）
     → 被告自身が巨大市場を認識し、積極的に拡大を求めていた
   ⑤逸失利益の算定:
     - 正昌 Order: ¥XX 万 × 30% = ¥XX 万（確実な逸失利益）
     - PM 報酬: 山口が合意（メール証拠）
     - 時間投入: 8 ヶ月 × 月額相当（フリーランスPM相場 80-150万/月）」

  ⑥さらに追加の逸失利益:
     - 璿元（顧客）: 発注予定だったが、欠陥製品のため代理を断念 → 失注
     - 八仙山標案（政府入札）: 機器がないため、測量を外注する羽目に → 持ち出し
     - 将来の政府標案: 既に最高レベルに参入済みだったが、
       販売できる機器がなくなった → 継続的な収益機会の喪失
     → これらは「希望」ではなく「既に成立していた商流の喪失」

  → 証拠: 正昌 Order Form、送金証明、12/10 メール、SOP メール、
          璿元とのやり取り（あれば）、八仙山標案の契約書/仕様書
  → 判定: 正昌 Order + 璿元の発注意向 + 八仙山標案 = 3 つの具体的実績。
          「単なる希望」ではなく「既に商流が動いていた」ことが立証可能。
          裁判官は具体的受注実績があれば逸失利益を認めやすい。
          認容額: ¥1,000-3,000 万（8ヶ月人件費含む）。
```

#### 反論⑤: 消滅時効・権利濫用

```
被告主張:
  「（時効は主張できない — 事故は 2026/2、提訴は 2026/4 → 2ヶ月しか経っていない）」
  「原告の請求額は事実上の懲罰的損害賠償であり、日本法では認められない。」

★ あなたの反撃:
  「①日本法で懲罰的損害賠償が認められないのは事実だが、
     私の請求は全て『実損害 + 逸失利益』の範囲内。
   ②詐欺的行為に対しては信義則上、制裁的要素を含む
     損害認定がされる判例がある（東京地判 H28 等）。」

  → 判定: 消滅時効は使えない。権利濫用も通らない。
```

## 第2ラウンド：裁判官の心証予測

```
裁判官が重視するポイント（優先順）:

1. 保証書の存在
   → 保証書がある = 完成品として販売 → 開発版の言い訳は通らない

2. Mapry 社員（宮田氏）が操縦して墜落
   → 製造者自身が使って壊れた = 過失相殺ゼロ

3. version "0.0.0" + bash_history
   → 開発版を ¥800 万で売った = 契約不適合は明白

4. 全バッテリー保護の無効化
   → 「意図的に」安全機能を切った = 重過失 or 故意

5. 山口のメール（市場認識 + PM 承認）
   → 逸失利益の予見可能性を被告自身が証明してくれている
```

### 裁判官の和解勧告（予測）

```
「被告は機体代金 ¥800 万の返還を認めますか？」
→ Mapry: 「...認めます」（ここは争っても勝てない）

「では残る争点は損害額ですね。」

裁判官の心証:
  機体返金:     ¥800 万（確定）
  人件費:       ¥500-800 万（8ヶ月 × 相当額、一部減額）
  逸失利益:     ¥1,000-2,500 万
    内訳:
    - 正昌 Order 30% 手数料（実績）
    - 璿元 失注分（発注意向があった）
    - 八仙山標案 外注費持ち出し（政府契約 = 金額確定）
    - 将来の政府標案機器販売（既に参入済み = 蓋然性高）
    - PM 報酬（山口が承認済み）
  慰謝料:       ¥50-200 万（詐欺的要素 + 信用毀損加算）
  ──────────────────
  和解提示額:   ¥2,500-3,500 万

  ★ 被告が渋れば裁判官:
  「被告が和解に応じない場合、判決になりますが、
   判決では利息（年3%）も加算されますよ。
   また、この証拠内容では被告に不利な判決が予想されます。」
```

## 第4ラウンド：交渉戦術

### あなたが ADR で使うべきカード（順番が重要）

```
第1手: 技術証拠のみ提示（甲9号証の1 = 31項目）
  → 「これが事実です」（感情を排除、淡々と）
  → 相手に衝撃を与えるが、まだ全弾は出さない

第2手: メール交差立証（EMAIL_ADR_EVIDENCE.md の内容）
  → 「山口社長が市場の巨大さを認識していたこと」
  → 「同時にテスト中の不安定品を出荷したこと」
  → 「事故後に報酬を拒否したこと」
  → 相手: 返す言葉がない

第3手: 和解金額の提示
  → ¥3,000 万（高めに出す）
  → 相手が「高すぎる」と言ったら:

第4手: 行政通報カードを暗示（直接言わない）
  → 「和解が成立しない場合、他の手続きも検討せざるを得ません」
  → 「航空法上の報告義務についても確認しているところです」
  → 相手弁護士: （国交省通報の暗示だと理解する）

第5手: 相手が減額交渉してきたら
  → ¥2,000-2,500 万で和解成立
  → 守秘義務条項付き（Mapry が面子を保てる）
  → 行政通報しない旨の確約
```

### ❌ やってはいけないこと

```
1. 感情的になる
   → 裁判官/ADR 委員は冷静な人に味方する

2. 最初から全カードを出す
   → 相手に反論準備の時間を与えてしまう

3. 刑事告訴を先にちらつかせる
   → 「恐喝」と言われるリスク（実際にはならないが印象が悪い）
   → 行政通報の暗示にとどめる

4. SNS で公開する（今は）
   → 和解前に公開すると相手が態度を硬化させる
   → 和解成立後 or 判決後に匿名で公開（LegalShield のケーススタディに）

5. ¥800 万で妥協する
   → あなたの証拠力で ¥800 万は安すぎる
   → 最低 ¥1,500 万、目標 ¥2,500 万
```
