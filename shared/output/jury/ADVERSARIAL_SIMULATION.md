```
   - 機体代金: ¥800 万
   - 車両損傷: ¥XX 万
   - 逸失利益: SOP 30% × 台湾市場予想 + PM 報酬
```

#### 反論③: 因果関係の否定

```
被告主張:
  「欠陥と墜落の因果関係が不明確。
   磁力計異常があったとしても、それが直接の墜落原因とは限らない。

★ あなたの反撃:
  「①出荷前 syslog に 4,509 回の磁力計異常 → 台湾に来る前から異常
   ②日本でのテスト中にも Battery Failsafe 発動 (12/10 syslog.1)

     PL 法では『どの欠陥が原因か』の特定は不要。
     『欠陥があった + 損害が生じた』で足りる（PL法§3の立証構造）。」

  → 証拠: syslog 全文、パラメータファイル、フォレンジック報告
  → 判定: PL法の立証構造上、個別因果関係の特定は不要。被告の反論は空振り。

```

```
★ あなたの反撃:
  「①正昌からの Order Form（実際の受注 = 実績）
   ②正昌からの送金証明（実際に代金が動いている）
   ③山口が 12/10 に PM 契約 + 鴻海/台塑連携を承認（メール証拠）
     → 被告自身が市場の実現性を認めていた
   ④EV・ロボット・トラックへの展開も山口が要望（12/10 返信）
     → 被告自身が巨大市場を認識し、積極的に拡大を求めていた
   ⑤逸失利益の算定:
     - 正昌 Order: ¥XX 万 × 30% = ¥XX 万（確実な逸失利益）
     - PM 報酬: 山口が合意（メール証拠）

  ⑥さらに追加の逸失利益:
     - 璿元（顧客）: 発注予定だったが、欠陥製品のため代理を断念 → 失注

  → 証拠: 正昌 Order Form、送金証明、12/10 メール、SOP メール、
          璿元とのやり取り（あれば）、八仙山標案の契約書/仕様書
  → 判定: 正昌 Order + 璿元の発注意向 + 八仙山標案 = 3 つの具体的実績。
          「単なる希望」ではなく「既に商流が動いていた」ことが立証可能。
          裁判官は具体的受注実績があれば逸失利益を認めやすい。
          認容額: ¥1,000-3,000 万（8ヶ月人件費含む）。
```

## 第5ラウンド：判例に基づく法的根拠の補強

### 5.1 PL 法（製造物責任）の判例法理

```
■ 最判平成25年3月21日（イレッサ訴訟・上告審）
  要旨: PL法の「欠陥」は設計上・製造上・指示警告上の3類型。
  あなたのケース:
  - 設計上の欠陥: OA_TYPE=0（障害物回避を無効化して設計）
  - 製造上の欠陥: version 0.0.0（開発版のまま出荷 = 品質管理の完全欠如）
  - 指示警告上の欠陥: 31項目の欠陥を告知せず販売

■ 東京地判平成14年12月13日（ガス瞬間湯沸器事件）
  要旨: 安全装置を意図的に外して販売 → 製造者の重過失認定。
  あなたのケース:
  - BATT_CRT_VOLT=0, BATT_FS_CRT_ACT=0 → バッテリー保護を全て無効化
  - ARMING_CHECK → 磁力計チェックを無効化
  = 「安全装置を意図的に外した」のと同一構造。

■ 大阪地判平成6年3月29日（松下電池PL訴訟）
  要旨: 製造者がテスト段階で異常を認識していた場合、
        「通常有すべき安全性」を欠くことの立証が容易になる。
  あなたのケース:
  - bash_history: 出荷前に54回リブート = テスト段階の異常認識
  - syslog: 4,509回の磁力計異常
  = 「認識していた」は SD カードで完全に立証済み。
```

### 5.2 詐欺（民法§96）の判例法理

```
■ 最判昭和33年6月14日
  要旨: 取引上の信義則に基づく告知義務がある事項を故意に沈黙した場合、
        積極的欺罔と同視される（不作為の詐欺）。
  あなたのケース:
  - 31項目の安全欠陥を知りながら沈黙 = 不利益事実の不告知
  - 保証書を発行 = 完成品であるとの積極的表示
  - 同時期に PM 契約・鴻海連携を承認 = 市場の巨大さを認識しながら
  = 「告知すべき事項を故意に沈黙」+ 「積極的に完成品と表示」の二重構造。

■ 東京高判平成17年1月25日（投資用マンション欺罔事件）
  要旨: 売主が重要な事実を告知しなかった場合、
        買主の錯誤を利用した詐欺として取消可能。
        取消の効果 = 代金全額返還 + 損害賠償。
  あなたのケース:
  - 「林業測量用完成品」として ¥800 万で販売
  - 実際は version 0.0.0 の開発機
  = 代金の full refund + 損害賠償の法的根拠が明確。
```

### 5.3 逸失利益の判例法理

```
■ 最判昭和43年11月15日（富貴丸事件）
  要旨: 逸失利益は「相当程度の蓋然性」があれば認容される。
        確実性までは不要。

■ 東京地判平成18年1月30日
  要旨: 事業の逸失利益は「過去の実績」と「具体的な見込み」で算定。
        - 過去の実績がある場合: そのまま基準
        - 新規事業の場合: 具体的な受注見込み + 事業計画で算定

  あなたのケース — 逸失利益の立証構造:

  ┌─ 層1: 確実な実績 (裁判所が認めやすい) ──────────┐
  │  - 正昌 ORDER FORM (実際の受注)                   │
  │  - 正昌からの送金証明 (代金が動いている)          │
  │  - 璿元の発注意向 (あれば LINE 等の記録)          │
  │  - 八仙山標案 (政府入札 = 金額確定)               │
  └──────────────────────────────────────────────────┘
```

### 5.4 優越的地位濫用（独禁法）の判例法理

```
■ 公取委排除措置命令平成21年6月22日（セブン-イレブン事件）
  要旨: 取引上の地位が相手方に優越している事業者が、
        正当な理由なく不利益を与える行為は優越的地位の濫用。

  あなたのケース:
  - Mapry = 製品の唯一の供給者（代替手段なし）
  - あなた = 台湾唯一の代理店（Mapry に依存）
  - 威脅斷貨 = 「取引の継続が困難になる不利益」
  - 「成果物がないので報酬払えない」= 正当な理由なき不利益
  → 優越的地位の濫用の構成要件を満たす。

  ※ ただし: 独禁法の民事損害賠償請求（独禁法§25）は
    公取委の排除措置命令が前提。直接の損害賠償は民法§709 で請求する方が確実。
    「優越的地位濫用」は交渉カード（公取委に申告する姿勢）として使う。
```
